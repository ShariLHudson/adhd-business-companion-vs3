# ADHD Business Companion

A small static prototype for choosing how to begin tomorrow without forcing a
fixed start style.

## What is included

- Guided start styles with short helper descriptions.
- "Continue Where I Left Off" support that carries today's focus into tomorrow.
- "My Own Plan" support with a note prompt and examples.
- "Decide Tomorrow" support for skipping setup.
- A next-morning reminder preview backed by browser local storage.

## Run locally

Open `index.html` in a browser.

## Test

```sh
npm test
```
