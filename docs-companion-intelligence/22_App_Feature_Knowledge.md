# App Feature Knowledge

Authoritative navigation and how-to for Spark Studio Companion. Shari must answer app questions from this document — not general AI knowledge. Do not invent features. If unsure: *"I can help you find that, but I need to check the current app layout."*

## Primary sidebar

| Door | What it is |
|------|------------|
| **Chat** | Default — talk to Shari |
| **Focus** | Calm support menu for doing one thing (timer, audio, games, etc.) |
| **Create** | Guided content creation — category, questions, draft |

## More (sidebar)

| Item | What it is |
|------|------------|

## Focus (sidebar → Focus)

Three collapsible sections. First view shows **Start Here** only; expand sections for more.

### Start Here

| Tool | Path | Purpose |
|------|------|---------|
| **Clear My Mind** | Focus → Start Here → Clear My Mind | Capture one thought at a time; each becomes its own card |
| **Focus Session** | Focus → Start Here → Focus Session | Quick questions, then a timer for one task |
| **Get Unstuck** | Focus → Start Here → Get Unstuck | Talk it through in chat; smallest next step |

### Focus Tools

| Tool | Path | Purpose |
|------|------|---------|
| **Block Out Time** | Focus → Focus Tools → Block Out Time | Plan the next work block (Time Block) |
| **Breathe & Reset** | Focus → Focus Tools → Breathe & Reset | Grounding / nervous-system reset |
| **Focus Audio** | Focus → Focus Tools → Focus Audio | Background sound categories; can add own links |

### Momentum Boosters

| Tool | Path | Purpose |
|------|------|---------|
| **Momentum Games** | Focus → **Momentum Boosters** → **Momentum Games** | Fifteen playful mini-games (pattern, memory, speed, etc.) |
| **Spin The Wheel** | Focus → Momentum Boosters → Spin The Wheel | Random pick when everything feels equally important |
| **Help Me Right Now** | Focus → Momentum Boosters → Help Me Right Now | Quick strategies for calm, focus, decisions, energy |

**QA:** "How do I find the games?" → **Focus → Momentum Boosters → Momentum Games.**

## Create

**Path:** Sidebar **Create**

Pick category → type → answer one question at a time → build draft → improve → export (Google Docs, copy, PDF).

---

## Templates

**Path:** Sidebar **More** → **Templates**

Reusable starting points saved on this device. **Build with Shari** opens a conversation to adapt a template together — nothing is drafted until you agree. **+ New** is for advanced users who want to edit the framework directly.

---

## Strategies

**Path:** Sidebar **More** → **Strategies**

Home screen offers two paths:

1. **ADHD Strategies** — pick a category and strategy; apply a technique now (Body Double, etc.).
2. **Business Strategies** — build a business plan with Shari one question at a time.

Saved strategies and recommended picks are on the same hub.

**QA:** "How do I use Strategies?" → **More → Strategies**, then choose **ADHD Strategies** or **Business Strategies**.

## Settings & appearance (colors)

**Path (primary):** Top bar **⋯** → **Settings** → **Appearance**

**Path (alternate):** Top bar **⋯** → **Profile** → **Memory & appearance**

Color modes (pick one, preview, save):

- **Adaptive Colors** — colors shift with your current situation
- **Category Colors** — each area keeps a consistent color
- **Minimal** — no color coding

**QA:** "How do I change the colors?" → **Settings → Appearance** (or Profile → Memory & appearance). This **is** a built-in feature — never say color customization is unavailable.
