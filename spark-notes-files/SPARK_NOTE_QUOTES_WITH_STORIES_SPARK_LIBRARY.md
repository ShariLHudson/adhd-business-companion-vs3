# Spark Note™ Quotes With Stories Spark Library

## Purpose

This library contains meaningful quotes paired with the stories behind
them.

A quote by itself can be inspiring.

A quote with context creates connection.

Each Quote Spark should include:

-   who said it
-   what was happening in their life
-   why the words mattered
-   how the user can apply the idea

The goal:

Turn words into wisdom.

## Category

Quote

possible.

## Story

# Quote Spark Guidelines

Quote Sparks should:

-   provide context
-   tell the human story
-   explain meaning
-   create reflection

Avoid:

-   random inspirational quotes
-   quotes without sources
-   quotes that feel generic

The message:

Wisdom becomes more powerful when we understand the story behind it.
