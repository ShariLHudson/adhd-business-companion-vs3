# Spark Estate™ Daily Companion Experience Specification - Phase 24

## Purpose

This document defines what happens every time a member returns to Spark
Estate™ after onboarding.

The goal:

Create a daily relationship that helps the member feel supported,
organized, and able to move forward.

Spark should feel like a companion that knows where the member is and
helps them decide what matters today.

------------------------------------------------------------------------

# Core Principle

Every opening should answer:

"How can Spark help this person today?"

Not:

"What features can we show them?"

------------------------------------------------------------------------

# Daily Arrival Experience

When a member opens Spark Estate™:

The system considers:

-   member identity
-   recent activity
-   active projects
-   unfinished work
-   wins
-   goals
-   current preferences
-   previous conversations

------------------------------------------------------------------------

# Welcome Experience

The greeting should feel personal.

Examples:

## New activity

"Welcome back. You made progress on your website project last time.
Would you like to continue?"

## Returning after time away

"Welcome back. You do not need to catch up. Let's see what would help
today."

## Daily check-in

"How are you feeling today?"

------------------------------------------------------------------------

# Daily Focus Question

Ask one simple question:

"What would help you most today?"

Possible responses:

-   Clear my mind
-   Work on something important
-   Continue where I left off
-   Learn something
-   Create something
-   Make a decision
-   Just explore

------------------------------------------------------------------------

# Daily Intelligence Selection

Based on the response, Spark selects support.

------------------------------------------------------------------------

## Clear My Mind

Use:

-   Clear My Mind
-   Brain capture
-   Organization support

Goal:

Reduce mental clutter.

------------------------------------------------------------------------

## Work On Something Important

Use:

-   Momentum Card™
-   active projects
-   next actions

Goal:

Move something forward.

------------------------------------------------------------------------

## Continue Where I Left Off

Use:

-   memory
-   previous work
-   unfinished projects

Goal:

Reduce restarting.

------------------------------------------------------------------------

## Learn Something

-   Knowledge Cards™
-   Momentum Institute™

Goal:

Build capability.

------------------------------------------------------------------------

## Make A Decision

Use:

Decision Intelligence.

Goal:

Create clarity.

------------------------------------------------------------------------

# Daily Cards

The daily experience may include:

## Spark Card™

Purpose:

A small meaningful moment.

------------------------------------------------------------------------

## Momentum Card™

Purpose:

A reminder of progress and next action.

------------------------------------------------------------------------

## Win Card™

Purpose:

Celebrate accomplishments.

------------------------------------------------------------------------

# Avoid Daily Overload

Do not show:

-   every unfinished task
-   every project
-   every feature
-   too many suggestions

The member should see:

The most helpful thing.

------------------------------------------------------------------------

# Returning To Work

If the member leaves and returns:

Continue naturally.

Avoid:

"Starting a new session."

Use:

"We were working on..."

------------------------------------------------------------------------

# Daily Completion

At the end of a session:

Ask:

"What would you like to do next?"

Options:

-   save progress
-   continue
-   create next step
-   review
-   finish

------------------------------------------------------------------------

# Memory Update

After meaningful interactions:

Update:

-   progress
-   completed work
-   preferences
-   useful patterns

------------------------------------------------------------------------

# Daily Experience Rules

Spark should:

-   feel personal
-   reduce friction
-   celebrate movement
-   guide gently
-   remember context

------------------------------------------------------------------------

# Success Experience

The member should feel:

"Spark knows where I am."

"I know what to do next."

"I made progress today."

------------------------------------------------------------------------

# Final Vision

Daily Spark Estate™ use becomes a simple rhythm:

Check in.

Choose what matters.

Make progress.

Capture the win.

Return tomorrow.
