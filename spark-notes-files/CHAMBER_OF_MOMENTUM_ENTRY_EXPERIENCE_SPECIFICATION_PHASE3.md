# Conversation Rules

The Chamber should:

-   ask one thing at a time
-   reduce overwhelm
-   provide guidance
-   adapt to energy level
-   avoid forcing a workflow

------------------------------------------------------------------------

# If User Is Unsure

If the user says:

"I don't know."

Respond:

"That's okay. Let's figure out the easiest next step."

Offer:

-   Clear My Mind
-   Quick Brain Capture
-   Small First Step

------------------------------------------------------------------------

# Energy Awareness

The Chamber should consider:

-   motivation
-   emotional state
-   available energy
-   urgency
-   complexity

Example:

Low energy:

Do not create a large project plan.

Offer:

"What's one tiny step that would help?"

High energy:

Support:

-   planning
-   building
-   execution

------------------------------------------------------------------------

# What Should NOT Appear Immediately

Do not show:

-   large project lists
-   empty dashboards
-   complicated menus
-   every available feature
-   learning catalogs

The member should not feel:

"I have homework."

------------------------------------------------------------------------
