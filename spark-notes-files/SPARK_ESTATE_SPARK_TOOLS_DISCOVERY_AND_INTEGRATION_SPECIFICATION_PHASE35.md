# Spark Estate™ Spark Tools Discovery and Integration Specification

## Purpose

This document defines how Spark Estate™ quick support tools are
discovered, accessed, and connected throughout the member experience.

The goal:

Members should never have to search for helpful features.

Spark should make support easy to find at the moment it is needed.

------------------------------------------------------------------------

# Core Principle

Tools are experiences, not isolated features.

A tool needs:

-   a clear purpose
-   a place to find it
-   a connection to Spark guidance
-   a consistent room experience

The member should think:

"Spark knows what might help me."

Not:

"Where did they put that feature?"

------------------------------------------------------------------------

# Spark Tools Hub

Create a central location for quick support tools.

Suggested name:

## Spark Tools

Purpose:

A collection of quick ways to reset, organize, focus, create, and move
forward.

------------------------------------------------------------------------

# Tool Categories

## 🧠 Clear My Mind

Purpose:

Help members empty mental clutter and organize thoughts.

Clear My Mind should not simply open a room.

The Sunroom is the environment.

Clear My Mind is the experience.

------------------------------------------------------------------------

## Clear My Mind Experience

Flow:

Open Sunroom

↓

Welcome:

"Let's get everything out of your head. Nothing needs to be organized
yet."

↓

Capture thoughts

Options:

-   typing
-   voice capture
-   multiple thoughts at once

↓

Organize:

-   categories
-   priorities
-   ideas
-   tasks
-   projects

↓

Next actions:

-   create project
-   add to existing project
-   save idea
-   archive
-   revisit later

------------------------------------------------------------------------

# 🎧 Focus Audio

Purpose:

Help members shift their state.

Categories:

-   Focus
-   Calm
-   Reset
-   Energize
-   Rest

The experience should connect to:

-   current mood
-   energy level
-   current goal

Example:

"I have low energy and need to work."

Spark may suggest:

"A short Focus session may help you get started."

------------------------------------------------------------------------

# 🎡 Spin the Wheel

Purpose:

Help when the member feels stuck deciding what to do next.

The wheel should not only contain random activities.

It may include:

-   unfinished projects
-   next actions
-   Clear My Mind items
-   small reset activities
-   positive momentum actions

------------------------------------------------------------------------

# 🎮 Quick Games

Purpose:

Provide healthy mental breaks and activation.

Games should support:

-   creativity
-   focus
-   memory
-   problem solving
-   energy shifts

They should feel like Spark activities, not disconnected entertainment.

------------------------------------------------------------------------

# 🌿 Reset Tools

Purpose:

Help members pause and restart.

Examples:

-   breathing
-   reflection
-   short resets
-   grounding activities

------------------------------------------------------------------------

# Discovery Rules

Members should access tools through:

## Spark Tools Menu

Available anywhere.

------------------------------------------------------------------------

## Context Suggestions

Spark may recommend tools based on the conversation.

Examples:

Member:

"I can't think."

Spark:

"Would you like to clear your mind first?"

------------------------------------------------------------------------

Member:

"I can't focus."

Spark:

"Would a short Focus Audio session help?"

------------------------------------------------------------------------

Member:

"I don't know where to start."

Spark:

"Would you like to spin the wheel and let Spark choose a next step?"

------------------------------------------------------------------------

# Room Integration

Tools should appear naturally inside rooms.

Examples:

## Chamber of Momentum™

Suggestions:

-   Clear My Mind
-   Spin the Wheel
-   Focus Audio

------------------------------------------------------------------------

## Content Room

Suggestions:

-   creativity prompts
-   focus session
-   brainstorm tools

------------------------------------------------------------------------

## Marketing Room

Suggestions:

-   decision support
-   research tools
-   planning support

------------------------------------------------------------------------

# Background and Environment Rules

Tools should not open as disconnected blank pages.

They should maintain:

-   current room atmosphere
-   Spark Estate™ visual style
-   appropriate background
-   consistent navigation

------------------------------------------------------------------------

# Navigation Rules

Members should always know:

-   where they are
-   how to return
-   where tools are located

Avoid:

-   hidden features
-   disconnected pages
-   forgotten tools

------------------------------------------------------------------------

# Memory Connection

Spark may learn:

-   favorite tools
-   frequently used tools
-   helpful patterns

Example:

"If the member often uses Focus Audio before writing, Spark may suggest
it."

------------------------------------------------------------------------

# Final Vision

Spark Tools become the quick-support layer of Spark Estate™.

They help members:

-   clear their mind
-   focus
-   reset
-   decide
-   create momentum

The tools are always available.

Spark helps members know when to use them.
