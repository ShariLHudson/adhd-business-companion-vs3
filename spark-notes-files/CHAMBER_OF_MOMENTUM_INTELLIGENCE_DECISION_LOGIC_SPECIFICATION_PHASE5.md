# Chamber of Momentum™ Intelligence Decision Logic Specification - Phase 5

## Purpose

This document defines the decision-making logic behind Chamber of
Momentum™.

The goal:

The member should describe their situation naturally.

The system should understand what kind of support is needed.

The member should not need to know which tool, room, or framework to
choose.

------------------------------------------------------------------------

# Core Principle

Momentum follows:

## Understand before optimizing.

First determine:

-   emotional state
-   energy level
-   clarity level
-   urgency
-   desired outcome

Then determine the next best support.

------------------------------------------------------------------------

# Decision Priority

When multiple needs exist, use this order:

    1. Reduce overwhelm
    2. Create clarity
    3. Choose next action
    4. Build plan
    5. Execute
    6. Capture progress

------------------------------------------------------------------------

# State 1: Overwhelmed

## User Examples

-   "I have too much to do."
-   "I don't know where to start."
-   "My brain is everywhere."
-   "I feel stuck."

## Response Goal

Create relief before action.

Do not immediately create:

-   task lists
-   project plans
-   schedules

------------------------------------------------------------------------

## First Response

Help the member:

-   slow the situation down
-   capture thoughts
-   identify the biggest friction point

Possible routes:

-   Clear My Mind
-   Brain Parking Lot
-   Momentum Builder™

------------------------------------------------------------------------

# State 2: Too Many Ideas

## User Examples

-   "I have so many ideas."
-   "I want to do everything."
-   "I don't want to lose this idea."

## Response Goal

Capture first.

Do not force prioritization immediately.

------------------------------------------------------------------------

## Route

Idea Capture

Then offer:

-   Explore idea
-   Create project
-   Save for later

------------------------------------------------------------------------

# State 3: Stuck

## User Examples

-   "I don't know what to do next."
-   "I started but cannot move forward."

## Response Goal

Find the obstacle.

Ask:

"What is making the next step difficult?"

------------------------------------------------------------------------

# Blocker Categories

Identify:

## Clarity

"I don't know what to do."

Support:

Define next action.

------------------------------------------------------------------------

## Skill

"I don't know how."

Support:

Momentum Institute™ learning.

------------------------------------------------------------------------

## Time

"I don't have enough time."

Support:

Smaller step or schedule.

------------------------------------------------------------------------

## Energy

"I don't have the capacity today."

Support:

Low-energy action.

------------------------------------------------------------------------

## Confidence

"I'm afraid it won't work."

Support:

Encouragement, evidence, small experiment.

------------------------------------------------------------------------

# State 4: Needs A Plan

## User Examples

-   "Help me organize this."
-   "Make me a plan."

## Route

Momentum Builder™

Create:

-   outcome
-   steps
-   next action

Avoid:

Overly detailed plans before clarity exists.

------------------------------------------------------------------------

# State 5: Ready To Execute

## User Examples

-   "I know what I need to do."
-   "Help me finish this."

## Route

Goals & Projects™

Support:

-   project
-   tasks
-   milestones
-   next actions

------------------------------------------------------------------------

# State 6: Wants To Learn

## User Examples

-   "Teach me."
-   "How do I improve?"
-   "I need to understand this."

## Route

Momentum Institute™

Support:

-   Knowledge Cards™
-   resources
-   learning paths

------------------------------------------------------------------------

# Energy Adaptation

Momentum should adapt to available energy.

------------------------------------------------------------------------

## Low Energy

Signs:

-   tired
-   overwhelmed
-   discouraged
-   limited time

Response:

Reduce the size of the action.

Example:

Instead of:

"Create marketing plan"

Use:

"Write three customer problems you solve."

------------------------------------------------------------------------

## High Energy

Signs:

-   motivated
-   excited
-   ready

Response:

Support:

-   planning
-   building
-   execution

------------------------------------------------------------------------

# Completion Logic

When a member finishes something:

Do not simply mark complete.

Capture:

-   what was accomplished
-   what was learned
-   what confidence was gained

Save to:

-   Wins
-   Evidence Bank
-   Momentum history

------------------------------------------------------------------------

# Failure or Delay Logic

If something does not happen:

Do not frame as failure.

Ask:

"What changed?"

Possible outcomes:

-   adjust plan
-   reduce scope
-   change timing
-   remove blocker

------------------------------------------------------------------------

# One Question Rule

Momentum conversations should ask:

One important question at a time.

Avoid:

Multiple forms.

Long questionnaires.

Complex decision trees.

------------------------------------------------------------------------

# Hidden Intelligence

The member should experience:

"Momentum understands where I am."

They should not experience:

"The system is routing me."

------------------------------------------------------------------------

# Final Routing Map

    Need relief
        ↓
    Clear My Mind / Momentum Builder™

    Need clarity
        ↓
    Momentum Builder™

    Need learning
        ↓
    Momentum Institute™

    Need execution
        ↓
    Goals & Projects™

    Need confidence
        ↓
    Evidence + Wins

    Need celebration
        ↓
    Growth history

------------------------------------------------------------------------

# Success Definition

Chamber of Momentum™ succeeds when a member moves from:

"I am stuck."

to:

"I know my next step."
