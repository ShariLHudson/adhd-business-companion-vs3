# Spark Note™ Gratitude and Meaning Spark Library

## Purpose

This library contains Sparks designed to create moments of gratitude,
reflection, connection, and meaning.

These Sparks explore universal human experiences:

-   appreciation
-   kindness
-   relationships
-   purpose
-   memories
-   small moments
-   personal meaning
-   making a difference

The goal:

Help users pause and notice the meaningful parts of life.

These Sparks should be welcoming to people from different:

-   cultures
-   countries
-   backgrounds
-   traditions
-   personal beliefs

They focus on shared human experiences rather than specific religious
teachings.

# Gratitude and Meaning Spark Guidelines

These Sparks should:

-   create reflection
-   encourage appreciation
-   highlight connection
-   respect different backgrounds
-   focus on universal experiences

Avoid:

-   assuming one belief system
-   preaching
-   telling users how they should feel
-   forced positivity

The message:

Meaning can be discovered in many different ways.
