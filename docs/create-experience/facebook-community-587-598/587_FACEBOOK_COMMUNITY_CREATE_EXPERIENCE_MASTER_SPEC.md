# 587 — Create a Facebook Community
## Master Experience Specification

## Mission
Build **Create a Facebook Community** as a complete guided Spark Estate creation experience that takes a user from an early idea to a positioned, branded, configured, launched, managed, and growing Facebook group.

This is not a Facebook post generator. It combines strategy, naming, branding, setup, welcome, content, launch, growth, moderation, measurement, operations, and team handoff.

## Primary outcome
The user leaves with:
1. Community Foundation Brief
2. Selected name and tagline
3. Audience, purpose, promise, and positioning
4. Facebook setup workbook
5. Banner and profile-image direction
6. About description
7. Rules and membership questions
8. Welcome and onboarding kit
9. Content and programming plan
10. Launch plan
11. Growth and retention plan
12. Moderator handbook
13. Analytics plan
14. Community operating manual
15. Optional Create-to-Project handoff

## Entry points
- Marketing Chamber
- Create
- Founder Studio
- Client Relationships
- Content
- Events
- Business Estate
- Natural requests such as “Help me create a Facebook group.”

## Canonical ownership
- Marketing: positioning, launch, acquisition, growth, measurement strategy
- Client Relationships: trust, belonging, member journey, retention, feedback
- Content: editorial pillars, recurring programming, content calendar
- Creative Studio: banner, visual identity, profile image, layout
- Events: lives, workshops, challenges, launch events
- Create: guided creation, artifacts, save/resume, editing, export
- Projects: tasks, dates, dependencies, owners, implementation
- Momentum: beginning, continuing, returning, finishing

## Experience principles
- One meaningful decision at a time
- Explain why each decision matters
- Recommend without taking control
- Never promise growth
- Preserve decisions and support resume
- Allow skipped sections
- Do not force monetization or daily posting
- Adapt to client, lead, peer, learning, paid, local, or interest communities
- Distinguish community health from vanity metrics
- Support ADHD-friendly pacing

## Phases
Discover → Position → Name → Design → Configure → Welcome → Seed → Launch → Grow → Engage → Moderate → Measure → Operate → Improve

## Completion states
idea-captured, foundation-defined, identity-selected, setup-ready, visual-ready, launch-ready, launched, operating, review-due, paused, archived

## Explicit Create rule
The platform may recommend this experience but must not begin, save, or convert it into a Project until the user intentionally chooses.
