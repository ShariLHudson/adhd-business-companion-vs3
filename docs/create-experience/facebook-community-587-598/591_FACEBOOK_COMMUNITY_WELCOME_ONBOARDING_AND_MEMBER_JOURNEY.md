# 591 — Welcome, Onboarding & Member Journey

## Journey
Discover → evaluate → request → approve → orient → introduce → participate → receive value → connect → contribute → return → advocate.

## Welcome kit
- welcome message
- pinned Start Here post
- orientation
- rules summary
- introduction prompt
- first-week action
- navigation guide
- help instructions
- optional welcome-video script

## Start Here structure
1. Welcome
2. Who this is for
3. What members can expect
4. What it is not
5. How to participate
6. Where to begin
7. Rules
8. How to get help
9. First introduction

## Better introduction prompts
- What brings you here?
- What are you working toward?
- What would you be glad to receive help with?
- What experience might you enjoy sharing?
- What would make this valuable?

## First seven days
Day 0 welcome; Day 1 introduction; Day 2 quick win; Day 3 discussion; Day 4 connection; Day 5 live/recorded welcome; Day 6 member story; Day 7 reflection and next step. Adapt this to capacity rather than creating posting pressure.

## Retention moments
First response, useful resource, connection, contribution, recognition, event, milestone, and invitation to help shape the community.

## Output
Welcome post, Start Here post, video script, introduction post, first-week journey, FAQ, expectations, help instructions, and re-entry message.
