# 589 — Naming, Brand & Banner System

## Naming patterns
- audience + outcome
- identity + community
- problem + progress
- profession + support
- location + interest
- movement + belonging
- branded program + clear descriptor
- founder brand + descriptive benefit

## Name scoring
Clarity, relevance, memorability, search usefulness, emotional resonance, distinctiveness, credibility, longevity, pronunciation, brand fit, and future flexibility.

## Avoid
Unclear cleverness, exaggerated claims, trend dependence, unexplained acronyms, competitor confusion, and names so broad they attract everyone.

## Tagline patterns
- A community for [audience] who want to [outcome] without [friction].
- Learn. Connect. [Progress verb].

## Visual outputs
- banner concept
- banner copy
- mood and imagery
- profile-image direction
- colors and typography
- icon/symbol direction
- mobile-safe layout
- image-generation prompt
- Canva-ready brief
- alt-text suggestion

## Current cover guidance
Keep dimensions configurable because Facebook may change them. The current official recommendation used for this build is 1,640 × 856 pixels, approximately 1.91:1. Keep names, faces, and essential copy near the center because mobile views can crop outer regions.

## Banner hierarchy
1. Group name
2. Short promise/tagline
3. Optional belonging cue
4. Minimal parent-brand identifier

Avoid dense text, schedules, URLs, lists, and essential information near edges.

## Output package
10–20 names, shortlist matrix, final name, tagline, About headline, banner copy, banner prompt, profile prompt, Canva brief, and accessibility note.
