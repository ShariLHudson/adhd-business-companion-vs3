# 136A — Fundraiser / Gala Domain Specification

A successful fundraiser strengthens relationships while advancing a mission and achieving measurable fundraising outcomes.

Domains:
- Donor experience
- Storytelling
- Sponsorship
- Giving journey
- Auctions
- Stewardship
- Accessibility
- Production
- Financial accountability
- Post-event cultivation

Formats:
- Gala
- Benefit dinner
- Charity auction
- Giving day event
- Virtual fundraiser
- Hybrid fundraiser

KPIs:
funds raised, donor retention, new donors, sponsorship revenue, attendance, auction performance, pledge fulfillment, follow-up engagement.
