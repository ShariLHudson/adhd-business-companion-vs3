## Governing distinction

Spark Estate can help create an extraordinary range of business work — but it must always distinguish between:

1. **fully_create** — Spark can fully create the object/content (Create Artifact SoT)
2. **prepare** — Spark prepares structure/draft; member completes
3. **user_provided** — requires Business Profile / DNA / avatar / offer context (273–278)
4. **completed_elsewhere** — Project execution, payments, signed agreements, external sync

## Production triad

A Blueprint is production-ready only when it passes:

1. Blueprint Createability Certification (233–236)
2. Blueprint Context Connection Certification (273–278)
3. Shared Object Library Certification (295–300)
