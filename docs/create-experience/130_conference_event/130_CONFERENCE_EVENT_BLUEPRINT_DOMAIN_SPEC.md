# 130A — Conference Domain Specification

A successful conference creates learning, networking, inspiration, and measurable outcomes at scale.

Domains:
- Program design
- Speaker management
- Sponsor experience
- Exhibitor operations
- Registration
- Attendee journey
- Accessibility
- Production
- Safety
- Logistics
- Post-event engagement

Conference styles:
- Educational
- Trade
- Corporate
- Membership
- Leadership
- Technology
- Community
- Hybrid

KPIs:
attendance, engagement, sponsor satisfaction, exhibitor ROI, session ratings, networking, retention, revenue, follow-up.
