# Universal Create Command & Action Library Constitution

- Defines every command, menu item, toolbar control, keyboard shortcut, mobile action, confirmation, permission, and state transition in Create.
- No visible control may be decorative, ambiguous, duplicated without reason, or implemented differently across work types.
- Every command must define its Command ID, label, purpose, scope, enablement, permissions, confirmation, persistence, undo behavior, audit event, shortcut, mobile equivalent, and browser test.
