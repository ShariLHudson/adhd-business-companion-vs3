# 080 — Standard Operating Procedure Work Type Schema

## Work Type ID

`standard_operating_procedure`

## Default Full Workshop Map

1. SOP Title
2. Purpose
3. Scope
4. Owner
5. Trigger
6. Inputs
7. Tools and Access
8. Roles
9. Preparation
10. Step-by-Step Procedure
11. Decision Points
12. Exceptions
13. Quality Checks
14. Safety or Risk
15. Records and Evidence
16. Completion Definition
17. Review Schedule
18. Revision History

## Editor Requirements

Step-by-Step Procedure must support:

- numbered steps;
- substeps;
- checkboxes;
- screenshots or attachments;
- warnings;
- responsible person;
- expected result;
- reorder.

## Completion

An SOP remains editable after approval or completion.

Changes must preserve revision history and allow a new effective version.
