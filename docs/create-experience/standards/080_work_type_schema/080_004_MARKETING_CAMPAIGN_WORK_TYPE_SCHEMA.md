# 080 — Marketing Campaign Work Type Schema

## Work Type ID

`marketing_campaign`

## Default Full Workshop Map

1. Campaign Purpose
2. Offer
3. Audience
4. Problem and Desired Outcome
5. Positioning
6. Core Message
7. Campaign Goal
8. Success Measures
9. Channels
10. Content Themes
11. Assets
12. Calls to Action
13. Funnel or Journey
14. Schedule
15. Budget
16. Responsibilities
17. Tracking
18. Testing
19. Risks
20. Launch Readiness
21. Performance Review
22. Lessons Learned

## Contextual Assistance

“Give Me Ideas” inside Channels must produce channel ideas, not generic campaign concepts.

“I’m Not Sure” inside Success Measures must explain possible measures and recommend a starting set.

## Project Conversion

Sections may become:

- campaign milestones;
- content tasks;
- asset-production tasks;
- review checkpoints;
- measurement tasks.
