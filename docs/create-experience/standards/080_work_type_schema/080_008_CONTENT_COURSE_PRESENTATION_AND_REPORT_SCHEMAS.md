# 080 — Content, Course, Presentation, and Report Schemas

## Content Plan

Sections may include:

- purpose;
- audience;
- themes;
- channels;
- content types;
- cadence;
- calls to action;
- schedule;
- reuse;
- measurement.

## Course

Sections may include:

- learner;
- transformation;
- learning outcomes;
- prerequisites;
- curriculum;
- modules;
- lessons;
- exercises;
- assessments;
- resources;
- delivery;
- accessibility;
- launch;
- evaluation.

## Presentation

Sections may include:

- purpose;
- audience;
- central message;
- desired action;
- structure;
- opening;
- key points;
- examples;
- visuals;
- interaction;
- close;
- speaker notes;
- rehearsal.

## Report

Sections may include:

- purpose;
- audience;
- scope;
- questions;
- methodology;
- evidence;
- findings;
- analysis;
- recommendations;
- limitations;
- summary;
- appendices.

Every section inherits the same clickable, editable, saveable, complete-for-now, and reopenable behavior.
