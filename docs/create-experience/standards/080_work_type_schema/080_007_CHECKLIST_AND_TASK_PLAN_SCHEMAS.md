# 080 — Checklist and Task Plan Schemas

## Checklist Work Type ID

`checklist`

## Default Full Workshop Map

1. Checklist Purpose
2. User
3. Trigger
4. Preparation
5. Checklist Items
6. Exceptions
7. Completion Definition
8. Review

## Checklist Item Capabilities

Each item may include:

- title;
- description;
- required or optional;
- owner;
- order;
- dependency;
- evidence;
- completion state;
- notes.

## Task Plan Work Type ID

`task_plan`

## Default Full Workshop Map

1. Outcome
2. Starting Point
3. Tasks
4. Order
5. Dependencies
6. Owners
7. Time and Effort
8. Resources
9. Risks
10. Completion Definition
11. Review

## Project Conversion

Checklist items or task-plan items may become Project tasks while preserving source linkage.
