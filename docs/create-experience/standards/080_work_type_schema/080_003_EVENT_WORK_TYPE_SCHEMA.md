# 080 — Event Work Type Schema

## Work Type ID

`event_plan`

## Purpose

Support complete event design, preparation, execution, and follow-up from the perspectives of host, attendees, speakers, volunteers, vendors, sponsors, and operational teams.

## Default Full Workshop Map

1. Event Type
2. Purpose
3. Outcomes
4. Audience
5. Format
6. Date and Time
7. Venue or Online Platform
8. Budget
9. Revenue and Pricing
10. Program and Agenda
11. Speakers and Facilitators
12. Attendee Experience
13. Registration
14. Marketing
15. Sponsors
16. Vendors
17. Volunteers and Team
18. Equipment and Technology
19. Accessibility
20. Food and Hospitality
21. Swag and Materials
22. Risk and Contingencies
23. Run of Show
24. Communication Plan
25. Day-of Operations
26. Follow-Up
27. Evaluation
28. Final Review

## Example Section: Purpose

Must support:

- direct editing;
- starter language;
- “I’m Not Sure” options;
- examples by event type;
- audience-specific ideas;
- research;
- completion;
- reopening.

## Example Section: Audience

Contextual help should consider:

- who benefits;
- attendee needs;
- buyer versus participant;
- accessibility;
- experience level;
- relationship to the host;
- attendance barriers.

## Completion Rule

The Event may be marked Complete for Now with unresolved optional sections, but the platform must show acknowledged gaps.

## Project Conversion

The member may convert:

- the full Event into a Project;
- each operational section into a task group;
- Run of Show into timed tasks;
- unresolved decisions into assignments.
