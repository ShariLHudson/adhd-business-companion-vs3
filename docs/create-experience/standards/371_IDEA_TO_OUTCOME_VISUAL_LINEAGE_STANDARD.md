# 371 — Idea-to-Outcome Visual Lineage Standard

## Purpose

Let users see how a thought became meaningful work.

## Supported lineage

A thought may move through:

1. Clear My Mind
2. cluster
3. Parking Lot
4. conversation
5. decision
6. creation
7. Blueprint
8. Project
9. milestone
10. completed outcome
11. celebration or evidence

## Automatic lineage rule

When a new creation is made from an existing thought, conversation, file, decision, or Project, the platform should automatically create the lineage connection.

The user should not need to manually document where the creation came from.

## User questions supported

- “What happened to that idea?”
- “Where did this Project come from?”
- “What did I create from these notes?”
- “Which decisions led here?”
- “What have I completed from this?”
