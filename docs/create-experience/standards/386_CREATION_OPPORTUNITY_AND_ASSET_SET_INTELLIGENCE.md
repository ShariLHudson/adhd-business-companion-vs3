# 386 — Creation Opportunity & Asset-Set Intelligence

## Purpose

Recognize when separate existing creations could form a more useful reusable set.

## Opportunity examples

Existing:

- proposal
- pricing guide
- onboarding guide
- FAQ

Possible recommendation:

- Client Starter Set

Existing:

- event timeline
- packing list
- volunteer guide
- speaker brief

Possible recommendation:

- Event Delivery Set

Existing:

- house rules
- checkout instructions
- emergency information
- local recommendations

Possible recommendation:

- Guest Information Set

## Recommendation requirements

Explain:

- what already exists
- what belongs together
- what value the set would add
- whether anything new is required
- whether the set remains separate artifacts
- which artifact is the source of truth

## No forced packaging

A visual grouping or saved collection may be enough.

Do not create a new combined document unless it improves actual use.
