# Platform Certification V2

Defines the platform-wide certification required before Universal Creation is considered production-ready.

Certification must include:
- repository architecture review;
- capability registry validation;
- data migration validation;
- real-browser scenarios;
- accessibility testing;
- security testing;
- performance testing;
- recovery testing;
- cross-platform integration testing;
- telemetry verification;
- rollback rehearsal.

Final certification requires evidence, not assertions.
