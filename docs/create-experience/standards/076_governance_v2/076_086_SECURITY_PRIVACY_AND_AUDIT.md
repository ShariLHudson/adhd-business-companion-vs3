# Security, Privacy & Audit

Defines tenant isolation, permissions, protected content, data minimization, secure sharing, audit trails, and approval rules for consequential actions.

Required controls:
- least-privilege access;
- explicit external-sharing permissions;
- protected Founder Studio information;
- immutable audit events for important mutations;
- revocable access;
- secure deletion lifecycle;
- no cross-member context leakage;
- human approval for consequential external actions.
