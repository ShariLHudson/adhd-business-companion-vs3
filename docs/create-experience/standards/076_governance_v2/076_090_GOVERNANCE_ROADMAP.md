# Governance Roadmap

Defines the phased adoption roadmap.

Phase 1 — Foundation
- install constitutional files;
- create capability registry;
- inventory existing creation paths.

Phase 2 — Core Runtime
- adopt Work ID, routing, persistence, and workspace contracts.

Phase 3 — Shared Capabilities
- migrate thinking, research, output, and collaboration.

Phase 4 — Consumer Adoption
- migrate Projects, Chamber, Board, Founder Studio, Business Estate, Knowledge, and Cartography.

Phase 5 — Certification
- complete browser, accessibility, security, performance, and recovery testing.

Phase 6 — Continuous Governance
- monitor drift;
- review exceptions;
- evolve standards;
- preserve compatibility.
