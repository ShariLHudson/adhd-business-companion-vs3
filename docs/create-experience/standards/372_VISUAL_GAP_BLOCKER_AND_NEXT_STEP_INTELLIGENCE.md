# 372 — Visual Gap, Blocker & Next-Step Intelligence

## Purpose

Use visual relationships to reveal what needs attention without turning the experience into a dashboard.

## Gap detection

May identify:

- missing artifact
- missing decision
- missing owner
- missing date
- missing source information
- missing professional review
- missing external action
- disconnected work
- stale adaptation
- unsupported promise

## Next-step logic

Recommend one next step based on:

- user intent
- active branch
- urgency
- dependency
- creation status
- Project status
- energy mode
- external completion

## No red-wall rule

Do not fill the map with warnings.

Surface the highest-value blocker first.
