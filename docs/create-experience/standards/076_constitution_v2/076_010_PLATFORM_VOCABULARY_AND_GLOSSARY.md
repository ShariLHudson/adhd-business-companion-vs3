# 076 — Platform Vocabulary and Glossary

## Work
A meaningful member-owned artifact with a Work ID.

## Work ID
Immutable identifier for one work item.

## Original Request
The first meaningful request that caused the work to exist.

## Working Intent
The member’s current objective for the work.

## Classification
The authoritative work type controlling structure and behavior.

## Workspace
The member-facing environment for one Work ID.

## Section
A first-class component of work with identity, content, state, and history.

## Consumer
A platform destination that uses Universal Creation capabilities.

## Capability
A reusable platform-owned behavior.

## Extension
A compatible specialization of a shared capability.

## Derivative
A new linked Work ID created through conversion, duplicate, or branch.

## Project
The execution layer linked to source work.

## Research Write-Back
The act of inserting accepted research into work, notes, tasks, or Knowledge.

## Resume
Restoring a Work ID, context, active section, and next step.

## Pause
Safely stopping without completing or archiving.

## Complete for Now
A member-facing completion state that preserves future editability.

## Archive
Remove from active views while preserving work.

## Delete
Move to Trash by default.

## Branch
Create an alternative evolution with explicit lineage.

## Template
Reusable structure derived from or created for future work.

## Shari
The primary companion experience and orchestration voice.

## Chamber Member
Specialized expert contributor using shared creation contracts.

## Round Table Board
Multi-perspective decision-support capability.

## Present Intent
What the member is trying to do now.

## Continuity
The relationship between the current turn and existing work.

## Durable Persistence
Verified authoritative storage.

## Recovery Buffer
Temporary local protection for unsaved content.

## Certification
Evidence that behavior works in a real browser and production-like environment.
