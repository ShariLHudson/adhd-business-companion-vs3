# 077 — Universal Toolbar and Action Menu

## Primary Toolbar

Every working section should provide:

- Save
- Undo
- Redo
- Status
- Give Me Ideas
- I’m Not Sure
- Help Me Think
- Research
- Ask Shari
- More

## More Menu

May include:

- Show Examples
- Ask Chamber
- Ask Board
- Add Note
- Add Task
- History
- Duplicate Section
- Export Section
- Print Section
- Archive Section
- Delete Section where permitted

## Work-Level Actions

At the full-work level:

- Rename
- Save
- History
- Duplicate
- Branch
- Turn Into Template
- Turn Into Project
- Print
- Export
- Share
- Mark Complete for Now
- Reopen
- Archive
- Delete

## Control Integrity

Every visible control must have:

- a real handler;
- loading state;
- success state;
- error state;
- keyboard access;
- analytics event;
- browser certification.
