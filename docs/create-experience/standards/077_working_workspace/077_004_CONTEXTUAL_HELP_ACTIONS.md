# 077 — Contextual Help Actions

## Purpose

Define how support actions work inside each individual section.

## Required Actions

Every applicable section must support:

- Give Me Ideas
- I’m Not Sure
- Help Me Think
- Show Me Examples
- Recommend a Starting Point
- Research This
- Ask Shari
- Ask a Chamber Member
- Ask the Board
- Review This
- Improve This
- Simplify This

## Context Packet

Each action must receive:

- Work ID
- Work type
- Section ID
- Section title
- Current section content
- Audience
- Purpose
- Constraints
- prior decisions
- related completed sections
- member preferences
- current status

## Give Me Ideas

Must return ideas specifically for the current section.

Example:

Inside Event → Audience:

- possible primary audiences;
- segmentation options;
- invitation approaches;
- accessibility considerations;
- recommendation.

It must not return generic event ideas unrelated to the Audience section.

## I’m Not Sure

Must:

1. acknowledge uncertainty;
2. explain what the section is asking for;
3. offer a few concrete options;
4. recommend a likely starting point;
5. allow “Use this,” “Adapt,” or “Skip for now.”

It must not repeat the same question unchanged.

## Help Me Think

Must help the member reason through the section using one manageable question at a time.

## Show Me Examples

Examples must match the current work type, section, audience, and tone.

## Insert Behavior

Suggestions must support:

- Insert
- Replace selected text
- Add below
- Save to Ideas
- Compare
- Dismiss
