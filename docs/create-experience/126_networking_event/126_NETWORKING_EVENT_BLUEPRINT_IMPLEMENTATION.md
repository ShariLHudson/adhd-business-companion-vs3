## Blueprint promise

Help users plan and run a purposeful networking event that creates meaningful, comfortable, and useful connections rather than simply putting people in a room together.

The Blueprint must support:

- small informal gatherings
- structured business networking
- community networking
- member networking
- industry networking
- hosted introductions
- facilitated connection activities
- recurring networking events
- hybrid networking when supported by the Event Work Type

The experience should adapt rather than force every networking event into the same format.

## Core design principles

A strong networking event must intentionally design:

- who should meet
- why those connections matter
- how people will begin conversations
- how discomfort and exclusion will be reduced
- how the event will create value beyond exchanging business cards
- how hosts and facilitators will help
- how follow-up will continue the value
- how success will be measured

The Blueprint should prevent a common failure mode:

> inviting people, providing refreshments, and assuming meaningful networking will happen by itself.
