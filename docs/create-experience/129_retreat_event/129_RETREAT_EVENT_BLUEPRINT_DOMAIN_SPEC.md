# 129A — Retreat Domain Specification

A successful retreat creates transformation, reflection, connection, and implementation—not simply time away.

Primary planning domains:
- Environment
- Psychological safety
- Learning and reflection
- Community
- Rest and recovery
- Logistics
- Accessibility
- Risk management
- Follow-through

Adaptive retreat styles:
- Strategic
- Leadership
- Team
- Wellness
- Creative
- Faith/community
- Planning
- Client appreciation

Success metrics:
- participant engagement
- completion
- implementation plans
- relationship growth
- wellbeing
- feedback
- return intent
