# 131A — Summit Domain Specification

A successful summit aligns influential people around important ideas, decisions, partnerships, and future direction.

Domains:
- Strategic vision
- Executive experience
- Speaker quality
- Networking
- Sponsor value
- VIP experience
- Media & communications
- Accessibility
- Production excellence
- Follow-through

Formats:
- Executive
- Leadership
- Innovation
- Customer
- Partner
- Industry
- Virtual
- Hybrid

KPIs:
attendance, executive engagement, partnerships, strategic outcomes, sponsor satisfaction, media reach, follow-up actions, repeat participation.
