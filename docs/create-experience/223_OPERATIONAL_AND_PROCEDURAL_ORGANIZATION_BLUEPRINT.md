# 223 — Operational & Procedural Organization Blueprint

## Blueprint ID

`organizing.operational_procedural`

## Purpose

Help a business organize how work gets done through clear procedures, workflows, roles, handoffs, documentation, recurring rhythms, and management visibility.

## Primary outcomes

The user can:

- identify operational friction
- map current workflows
- clarify ownership
- simplify steps
- create SOPs
- organize recurring work
- improve handoffs
- reduce dependency on memory
- establish management rhythms
- maintain and improve procedures
