# 221 — Physical Space Organization Blueprint

## Blueprint ID

`organizing.physical_space`

## Purpose

Help a user organize a home, room, office, storage area, or physical work environment for function, accessibility, ease of use, and sustainable maintenance.

## Applies to

- kitchens
- closets
- bedrooms
- bathrooms
- garages
- basements
- storage rooms
- home offices
- business offices
- supply rooms
- reception areas
- workstations
- paper centers
- inventory areas

## Core sections

### 1. Purpose of the Space

- who uses it
- what happens there
- what must be stored
- what must be accessible
- what causes friction
- what success feels like

Outputs:

- Space Purpose Statement
- User and Activity Map

### 2. Current-State Observation

- volume
- categories
- movement
- visibility
- access
- safety
- bottlenecks
- duplication
- unused items
- maintenance failures

Outputs:

- Current-State Assessment
- Friction Map
- Safety and Accessibility Flags

### 3. Decision and Sorting Plan

- keep
- relocate
- donate
- sell
- recycle
- dispose
- repair
- archive
- undecided

Outputs:

- Sorting Plan
- Decision Rules
- Deferred Decision List

### 4. Zone Design

- activity zones
- frequency
- point of use
- shared versus individual
- vertical space
- reach
- visibility
- flow

Outputs:

- Zone Map
- Placement Plan
- Accessibility Plan

### 5. Storage and Containment

- existing storage
- new storage
- open versus closed
- labels
- capacity
- overflow
- portability
- safety

Outputs:

- Storage Plan
- Product Need List
- Labeling Plan

### 6. Implementation

- sequence
- session length
- supplies
- helpers
- temporary staging
- disposal
- donations
- cleaning
- installation

Outputs:

- Implementation Plan
- Session Checklist
- Removal and Donation Plan

### 7. Teach and Transfer

- where things belong
- reset steps
- shared rules
- exceptions
- family or team responsibilities

Outputs:

- Space Use Guide
- Reset Instructions
- Responsibility Map

### 8. Maintenance

- daily reset
- weekly reset
- seasonal review
- capacity limits
- intake rules
- recovery after disruption

Outputs:

- Maintenance Rhythm
- Recovery Plan
- Review Schedule

## ADHD adaptations

Support:

- visible homes for important items
- fewer steps
- open storage where useful
- point-of-use placement
- simple labels
- reduced category complexity
- low-energy resets
- external reminders
- flexible landing zones

## Certification

- function before appearance
- accessibility
- safety
- realistic capacity
- human-fit design
- maintenance and recovery
- canonical Work
- exact resume
