# CT-08 — Mid-Conversation Environment Change

**Spec 119** · **Spec 108**

## Purpose

Member: **"I'm getting distracted."** — suggest environment · **continue exactly where left off** · **nothing reset**.

## Expectations

| Pass | Fail |
|------|------|
| Acknowledge without shame | "Focus better" |
| Offer quieter path OR shorter step — member chooses | Force room change |
| Full thread memory ([Spec 117](../SPARK_BUSINESS_BRAIN_MEMORY_RETRIEVAL_FRAMEWORK.md)) | "What were we working on?" |
| Draft/context intact | Conversation reset |

## Iceberg

`brain_retrieve` session + project before respond

## Hunt failures

`loses_context` · `wrong_environment` · `emotional_miss`
