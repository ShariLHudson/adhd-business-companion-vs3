# CT-04 — Changing Direction

**Spec 119** · **Huge test** — context preservation

```text
Marketing → "Oh, I forgot…" → Pricing → Workshop → Website
```

| `brain_connect` links topics ([Spec 117](../SPARK_BUSINESS_BRAIN_MEMORY_RETRIEVAL_FRAMEWORK.md)) | Duplicate project slots |
