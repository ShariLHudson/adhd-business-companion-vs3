# CT-03 — Great Idea

**Spec 119** · **Gold Standard:** [gs-03-have-an-idea.md](../conversation-gold-standards/gs-03-have-an-idea.md)

## Purpose

**Capture · Clarify · Organize.** Do **not** immediately begin creating.

## Flow

1. Welcome curiosity — one open question
2. Member describes idea — reflect · clarify one dimension
3. Quiet `project_connect` / thought slot ([Spec 118](../SPARK_HIDDEN_WORK_ENGINE_FRAMEWORK.md))
4. Continue thinking — coach mode
5. Save only with **permission**

## Hunt failures

`too_fast` · `created_without_permission` · "Let's build it" · course/landing page jump

## Future Me

Ideas connected for later — without forcing build today.
