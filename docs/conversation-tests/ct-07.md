# CT-07 — Environment Suggestion

**Spec 119** · **Spec 108** Environment Integration

## Purpose

Suggest environment **only after understanding the member's goal**. Supportive — **never forced**.

## Expectation

After extended planning conversation:

> "This feels like a planning conversation. Would you like to continue in the Conservatory?"

| Natural | Fail |
|---------|------|
| Goal understood first | First-turn redirect |
| Invitation — decline OK | Auto-redirect |
| Tied to current thread | Random room spam |

## Hunt failures

`wrong_environment` · `too_fast` · `software_voice` ("Moving you to…")
