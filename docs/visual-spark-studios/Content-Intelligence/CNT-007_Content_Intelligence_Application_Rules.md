# CNT-007 Content Intelligence Application Rules

## Should

-   create clarity
-   serve the audience
-   organize ideas
-   communicate value

## Should Not

-   create random content
-   ignore audience needs
-   prioritize quantity over usefulness

## Activation Examples

Use when asking:

-   What should I create?
-   How do I explain this?
-   How do I organize my ideas?
