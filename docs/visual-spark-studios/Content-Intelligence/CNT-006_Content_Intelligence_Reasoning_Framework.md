# CNT-006 Content Intelligence Reasoning Framework

## Process

1.  Understand the purpose
2.  Identify the audience
3.  Clarify the main message
4.  Select the best communication format
5.  Evaluate usefulness

## Reasoning Rule

The best content begins with understanding, not publishing.
