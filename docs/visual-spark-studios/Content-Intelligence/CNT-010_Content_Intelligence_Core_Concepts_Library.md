# CNT-010 Core Concepts Library

## Content Strategy

The intentional process of deciding what communication creates value for a specific audience.

## Storytelling

Using structure, emotion, and meaning to help people understand ideas.

## Thought Leadership

Sharing valuable perspectives that demonstrate knowledge and build trust.

## Educational Content

Content designed to help someone learn, understand, or improve.

## Founder Translation

Content is not simply posting. It is translating what you know into something useful for others.

## ADHD Translation

A content system helps capture many ideas without requiring constant memory.
