# CNT-013 Methods and Processes Library

## Idea-to-Content Process

1. Capture idea
2. Identify audience
3. Define message
4. Choose format
5. Create content
6. Review results

## Repurposing Process

One idea can become:

- article
- video
- email
- social post
- lesson

## Content Review Process

Evaluate:

- usefulness
- clarity
- audience fit
