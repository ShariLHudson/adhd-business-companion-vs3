# CNT-015 Diagnostic Intelligence

## Purpose

Identifies content challenges.

## Signals

"I have ideas but never create."

Possible causes:

- no capture system
- unclear audience
- perfectionism

"I create content but nobody responds."

Possible causes:

- unclear message
- wrong audience
- weak connection

## Diagnostic Rule

Content problems are often message problems before creation problems.
