# CNT-005 Content Intelligence Framework Library

## Idea-to-Content Framework

Purpose: Turn ideas into usable content.

Steps:

1.  Capture the idea
2.  Define the audience
3.  Identify the message
4.  Choose the format
5.  Create and refine

## Storytelling Framework

Includes:

-   Situation
-   Challenge
-   Insight
-   Transformation

## Content Repurposing Framework

One core idea can become:

-   article
-   post
-   video
-   lesson
-   email

## ADHD Application

Creates structure so ideas do not remain unfinished.
