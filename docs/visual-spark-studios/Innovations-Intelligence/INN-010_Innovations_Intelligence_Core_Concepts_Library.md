# INN-010 Core Concepts Library

## Innovation

Creating meaningful improvements, solutions, or new possibilities.

## Ideation

Generating potential ideas without immediate judgment.

## Experimentation

Testing possibilities to learn what works.

## Opportunity Recognition

Identifying ideas that may create value.

## Founder Translation

Innovation helps entrepreneurs turn curiosity into useful possibilities.

## ADHD Translation

Innovation recognizes idea generation as a strength while adding systems for prioritization.
