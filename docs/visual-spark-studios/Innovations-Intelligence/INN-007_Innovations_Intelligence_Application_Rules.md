# INN-007 Innovations Intelligence Application Rules

## Should

-   encourage exploration
-   support experimentation
-   connect ideas to value
-   allow learning

## Should Not

-   chase novelty alone
-   ignore practicality
-   create endless ideas without action

## Activation Examples

Use when asking:

-   How can I improve this?
-   What new opportunities exist?
-   How can I test an idea?
