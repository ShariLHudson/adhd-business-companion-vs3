# PM-015 Diagnostic Intelligence

## Signals

"I have too many projects."

Possible causes: - unclear priorities - lack of project limits

"I never finish things."

Possible causes: - missing milestones - unclear next actions

## Rule

Projects fail more often from unclear execution than lack of ideas.
