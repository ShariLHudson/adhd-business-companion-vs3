# PM-009 Knowledge Boundary Map

## Purpose

Defines what Project Management Intelligence owns within Spark Estate™.

## Owns

-   project planning
-   execution structure
-   timelines
-   task coordination
-   resource organization
-   progress tracking

## Supports

Works with:

-   Strategy Intelligence for priorities
-   Momentum Intelligence for progress
-   Systems Intelligence for workflows
-   Creative Intelligence for execution planning

## Does Not Own

Project Management Intelligence does not own:

-   deciding business direction
-   replacing leadership
-   creating the vision

## Core Rule

Project Management turns ideas and goals into organized action.
