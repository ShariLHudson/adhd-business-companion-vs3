# PM-010 Core Concepts Library

## Project Management

The process of organizing people, resources, tasks, and timelines to
achieve an outcome.

## Scope

Understanding what is included and what is not included.

## Milestones

Important points that show progress toward completion.

## Dependencies

Things that must happen before other work can continue.

## Founder Translation

Project management creates the bridge between ideas and finished
results.

## ADHD Translation

External systems reduce overwhelm by making progress visible.
