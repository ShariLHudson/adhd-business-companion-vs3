# MKT-015 Diagnostic Intelligence

## Signals

"I create content but nobody responds."

Possible causes: - unclear audience - unclear message - wrong channel

"People do not understand my offer."

Possible causes: - positioning issue - value communication issue

## Rule

Marketing problems often begin with understanding before promotion.
