# MKT-007 Marketing Intelligence Application Rules

## Should

- understand people
- communicate value clearly
- support trust
- encourage consistency

## Should Not

- create pressure-based tactics
- ignore audience needs
- focus only on visibility

## Activation Examples

Use when asking:

- How do I market my business?
- Who is my audience?
- How do I explain my value?
