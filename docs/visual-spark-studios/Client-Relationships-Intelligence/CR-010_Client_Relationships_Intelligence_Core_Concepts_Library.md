# CR-010 Core Concepts Library

## Client Experience

The complete experience a client has before, during, and after working with a business.

## Trust

Built through:

- consistency
- communication
- reliability
- follow-through

## Retention

Keeping valuable relationships strong by continuing to create value.

## Client Success

Helping clients achieve the outcome they originally wanted.

## Founder Translation

A client relationship is not a transaction. It is an ongoing partnership.

## ADHD Translation

Systems and reminders can support consistent relationship maintenance.
