# SPARK-015 Personalized Action Framework

## Purpose

Defines how Spark converts understanding into useful next steps.

## Process

Understand → Identify Need → Choose Intelligence → Recommend Action →
Support Follow Through

## Principle

Actions should fit the person's current capacity, not an ideal version
of them.
