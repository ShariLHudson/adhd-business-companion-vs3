# SPARK-026 Data Ownership Model

## Purpose

Defines what information belongs to each intelligence layer.

## User Data

Includes: - profile - preferences - goals - patterns

## Experience Data

Includes: - interactions - feature usage - helpful responses

## Intelligence Data

Includes: - frameworks - knowledge - reasoning rules

## Principle

Data should support personalization while respecting boundaries.
