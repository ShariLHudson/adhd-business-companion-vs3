# SPARK-046 Framework Library Management

## Purpose

Defines management of models and frameworks.

## Each Framework Includes

-   name
-   purpose
-   creator/source
-   application
-   limitations

## Rules

Frameworks should: - solve a defined problem - have clear usage - avoid
unnecessary overlap

## Principle

Frameworks are thinking tools.
