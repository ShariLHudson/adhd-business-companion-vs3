# SPARK-044 Knowledge Retrieval Strategy

## Purpose

Defines how Spark finds relevant intelligence.

## Retrieval Steps

1.  Understand user intent
2.  Identify needed capability
3.  Activate relevant intelligence
4.  Gather supporting knowledge
5.  Create response

## Principles

Retrieve what helps. Avoid unnecessary complexity.

## Goal

The right knowledge at the right moment.
