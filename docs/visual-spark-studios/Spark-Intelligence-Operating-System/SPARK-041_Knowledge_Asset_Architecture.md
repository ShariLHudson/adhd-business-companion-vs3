# SPARK-041 Knowledge Asset Architecture

## Purpose

Defines how Spark organizes intelligence assets.

## Asset Types

-   Chamber Member knowledge
-   frameworks
-   templates
-   worksheets
-   reference materials
-   user-generated insights
-   experience assets

## Structure

Knowledge should have: - owner - purpose - boundaries - relationships -
usage rules

## Principle

Every asset should have a clear home and purpose.
