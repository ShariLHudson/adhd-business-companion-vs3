# SPARK-067 Accountability Framework

## Purpose

Defines supportive accountability.

## Includes

-   commitments
-   check-ins
-   progress reviews
-   obstacle identification

## Principles

Accountability should: - encourage ownership - create clarity - support
progress

It should not: - create shame - create pressure without support
