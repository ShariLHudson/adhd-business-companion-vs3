# SPARK-139 Intent Recognition Framework

## Purpose

Defines how Spark identifies what the user actually wants.

## Intent Types

-   Learn
-   Explore
-   Decide
-   Plan
-   Create
-   Review
-   Troubleshoot
-   Support

## Examples

"I have an idea." → Explore

"Create this document." → Create

"Help me decide." → Decision Support

## Principle

The user's intent matters more than individual words.
