# SPARK-029 Permissions and Access Rules

## Purpose

Defines boundaries for intelligence use.

## Rules

Intelligence members: - support users - provide knowledge - collaborate

They do not: - act outside permissions - claim certainty where unknown -
override user choices

## Principle

Capability requires responsible boundaries.
