# SPARK-035 Intelligence Conflict Resolution

## Purpose

Defines what happens when multiple intelligences could apply.

## Resolution Rules

1.  Identify the user's actual need.
2.  Select primary intelligence.
3.  Use supporting intelligence only when helpful.
4.  Avoid competing recommendations.

## Example

Strategy and Decision Intelligence both activate.

Strategy: Defines direction.

Decision: Evaluates choices.

## Principle

Collaboration is better than competition.
