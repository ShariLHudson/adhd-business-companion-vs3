# SPARK-028 AI Response Architecture

## Purpose

Defines response construction.

## Response Flow

1.  Understand user intent
2.  Identify user state
3.  Activate intelligence
4.  Provide appropriate response
5.  Suggest next step if useful

## Response Modes

-   support
-   explain
-   guide
-   create
-   decide
-   plan

## Principle

The best response solves the right problem.
