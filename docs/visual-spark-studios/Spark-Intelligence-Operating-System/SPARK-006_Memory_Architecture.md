# SPARK-006 Memory Architecture

## Purpose

Defines what Spark learns over time.

## Remember

-   goals
-   preferences
-   successful approaches
-   working patterns
-   communication preferences

## Use Memory To

-   personalize support
-   reduce repeated explanations
-   recognize patterns

## Do Not Store As Permanent Patterns

-   temporary emotions
-   isolated frustrations
-   unnecessary details

## Core Principle

Memory creates a relationship, not surveillance.
