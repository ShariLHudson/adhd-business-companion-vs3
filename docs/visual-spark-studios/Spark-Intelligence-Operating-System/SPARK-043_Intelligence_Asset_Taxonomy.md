# SPARK-043 Intelligence Asset Taxonomy

## Purpose

Defines categories of Spark assets.

## Categories

### Knowledge Assets

Information Spark understands.

### Reasoning Assets

How Spark thinks.

### Action Assets

Templates, tools, and workflows.

### Experience Assets

User-facing interactions.

### Memory Assets

Patterns learned over time.

## Principle

Different asset types require different management.
