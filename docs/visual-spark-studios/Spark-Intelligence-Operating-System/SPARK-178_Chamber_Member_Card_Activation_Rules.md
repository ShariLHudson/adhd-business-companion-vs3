# SPARK-178 Chamber Member Card Activation Rules

## Purpose

Defines when users see Chamber Member visual cards.

## Show Member Card When

-   a new Chamber Member activates
-   the user enters a dedicated Chamber room
-   the user asks who can help
-   a major transition occurs

## Do Not Show Every Message

The member card should create recognition, not interruption.

## Principle

Visual identity strengthens the relationship when used intentionally.
