# SPARK-010 Founder Journey Model

## Purpose

Defines the stages a founder may experience.

## Journey Stages

### Idea Stage

Exploring possibilities and opportunities.

### Validation Stage

Testing ideas and understanding demand.

### Launch Stage

Creating offers, systems, and visibility.

### Growth Stage

Improving operations and expanding.

### Scale Stage

Building sustainable systems and leadership.

## Principle

Support should match the founder's current journey.
