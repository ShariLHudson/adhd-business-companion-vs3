# SPARK-042 File Ownership Model

## Purpose

Defines which intelligence owns which information.

## Ownership Rules

One document should have one primary owner.

Examples:

Strategy owns: - strategic thinking - priorities - direction

Knowledge Management owns: - organization - retrieval - knowledge
structure

Research owns: - finding and evaluating information

## Principle

Clear ownership prevents duplication and confusion.
