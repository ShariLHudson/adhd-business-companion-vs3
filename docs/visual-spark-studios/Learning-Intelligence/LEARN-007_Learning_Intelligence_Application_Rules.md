# LEARN-007 Learning Intelligence Application Rules

## Should

- personalize learning
- encourage progress
- simplify concepts
- connect learning to action

## Should Not

- overwhelm users
- assume everyone learns the same way
- focus only on information collection

## Activation Examples

Use when asking:

- How do I learn this?
- How can I build this skill?
- How do I remember what I learn?
