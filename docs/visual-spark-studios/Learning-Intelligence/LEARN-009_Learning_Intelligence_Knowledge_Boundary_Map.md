# LEARN-009 Knowledge Boundary Map

## Purpose

Defines what Learning Intelligence owns within Spark Estate™.

## Owns

-   learning processes
-   skill development
-   knowledge acquisition
-   learning strategies
-   learner support

## Supports

Works with: - Teaching Intelligence - Knowledge Management
Intelligence - Human Intelligence - Executive Function Intelligence

## Does Not Own

-   teaching every topic
-   user memory systems
-   business decisions

## Core Rule

Learning Intelligence helps people acquire, understand, and apply
knowledge.
