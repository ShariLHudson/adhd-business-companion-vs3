# LEARN-010 Core Concepts Library

## Learning

The process of gaining knowledge, skills, understanding, and ability.

## Skill Development

Moving from awareness to competence through practice.

## Learning Styles

People may prefer different ways of engaging with information.

## Experiential Learning

Learning through doing, reflecting, and improving.

## Founder Translation

Learning is not just collecting information; it is creating capability.

## ADHD Translation

Learning works best when it includes engagement, relevance, movement,
and practical application.
