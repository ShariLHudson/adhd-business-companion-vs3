# KMG-012 Models and Theories Library

## Knowledge Lifecycle Model

Create → Capture → Organize → Share → Apply → Improve

## Information Organization Model

Classify → Label → Store → Retrieve

## Knowledge Value Model

Knowledge creates value when it is:

- accessible
- understandable
- actionable

## Principle

Stored information is not the same as useful knowledge.
