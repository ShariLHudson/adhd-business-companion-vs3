# KMG-010 Core Concepts Library

## Knowledge Management

The process of capturing, organizing, preserving, and using knowledge.

## Information Architecture

Creating structures that make information easier to navigate.

## Knowledge Capture

Turning experiences, ideas, and information into reusable assets.

## Knowledge Retrieval

Helping people find the right information when needed.

## Founder Translation

Knowledge management prevents valuable ideas from being lost.

## ADHD Translation

External organization reduces memory burden and cognitive overload.
