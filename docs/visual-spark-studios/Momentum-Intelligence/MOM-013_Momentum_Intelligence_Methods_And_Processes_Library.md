# MOM-013 Methods and Processes Library

## Restart Process

1. Identify where progress stopped
2. Remove judgment
3. Find the smallest next action
4. Restart movement

## Momentum Building Process

1. Choose priority
2. Define next step
3. Create support
4. Track progress

## Completion Process

Reduce complexity, create visibility, celebrate progress.
