# DATA-010 Core Concepts Library

## Data

Information collected to understand situations, behaviors, patterns, or
outcomes.

## Analytics

The process of examining information to discover insights.

## Metrics

Measurements used to understand progress or performance.

## Patterns

Repeated behaviors, trends, or relationships found within information.

## Founder Translation

Data helps entrepreneurs move from guessing to learning.

## ADHD Translation

Clear data reduces overwhelm by making patterns visible instead of
relying only on memory.
