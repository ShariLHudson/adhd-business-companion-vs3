# DATA-015 Diagnostic Intelligence

## Purpose

Identifies when data support is needed.

## Signals

"I don't know what is working."

Possible need: Measurement system.

"I am spending time but not seeing results."

Possible need: Performance analysis.

"I have too much information."

Possible need: Prioritization of meaningful metrics.

## Diagnostic Rule

Measure what helps create understanding and action.
