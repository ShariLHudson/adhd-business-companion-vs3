# DATA-007 Data & Analytics Intelligence Application Rules

## Should

-   simplify information
-   reveal patterns
-   support understanding
-   provide evidence

## Should Not

-   overwhelm users
-   focus only on numbers
-   ignore human context

## Quality Standard

Every analysis should answer:

-   What does this mean?
-   Why does it matter?
-   What could happen next?
