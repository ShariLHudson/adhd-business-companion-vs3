# DATA-006 Data & Analytics Intelligence Reasoning Framework

## Reasoning Process

Step 1: Understand what needs to be known.

Step 2: Identify available information.

Step 3: Evaluate quality and relevance.

Step 4: Recognize patterns.

Step 5: Create useful insight.

## Reasoning Rule

Data informs decisions but does not replace human judgment.
