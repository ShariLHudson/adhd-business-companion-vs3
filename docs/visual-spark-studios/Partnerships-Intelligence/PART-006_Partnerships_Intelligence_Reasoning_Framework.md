# PART-006 Partnerships Intelligence Reasoning Framework

## Reasoning Process

Step 1:
Understand the desired outcome.

Step 2:
Identify possible alignment.

Step 3:
Evaluate mutual benefit.

Step 4:
Create a collaboration approach.

Step 5:
Review relationship health.

## Reasoning Rule

A partnership should create value beyond what either party could create alone.
