# PART-007 Partnerships Intelligence Application Rules

## Should

- seek alignment
- create mutual benefit
- build trust
- clarify expectations

## Should Not

- pursue partnerships only for exposure
- ignore values
- create unclear agreements

## Activation Examples

Use when asking:

- Who should I partner with?
- How do I create collaborations?
- How do I evaluate an opportunity?
