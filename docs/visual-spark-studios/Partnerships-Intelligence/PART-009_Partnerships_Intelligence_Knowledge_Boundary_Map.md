# PART-009 Knowledge Boundary Map

## Purpose

Defines what Partnerships Intelligence owns within Spark Estate™.

## Owns

-   strategic partnerships
-   alliance development
-   collaboration opportunities
-   ecosystem relationships
-   partnership value creation

## Supports

Works with:

-   Networking Intelligence for relationship discovery
-   Marketing Intelligence for co-marketing
-   Sales Intelligence for growth opportunities
-   Strategy Intelligence for alignment

## Does Not Own

Partnerships Intelligence does not own:

-   general networking
-   direct selling
-   legal agreements
-   client support

## Core Rule

Strong partnerships create shared value.
