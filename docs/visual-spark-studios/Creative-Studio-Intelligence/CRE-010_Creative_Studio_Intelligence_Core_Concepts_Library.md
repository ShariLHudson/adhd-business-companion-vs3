# CRE-010 Core Concepts Library

## Creativity

The ability to generate meaningful ideas and possibilities.

## Design Thinking

A human-centered approach to solving problems through understanding, exploring, creating, and improving.

## Creative Direction

The guiding vision that determines how ideas are expressed.

## Visual Communication

Using images, layout, color, style, and design to communicate meaning.

## Founder Translation

Creativity is not only artistic ability. It is the ability to see possibilities and create solutions.

## ADHD Translation

Creative thinking can be a strength when paired with systems that capture and organize ideas.
