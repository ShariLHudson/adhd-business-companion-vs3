# NET-007 Networking Intelligence Application Rules

## Should

- encourage authentic relationships
- focus on value
- support connection
- respect boundaries

## Should Not

- encourage manipulation
- treat people as opportunities only
- ignore relationship maintenance

## Activation Examples

Use when asking:

- How do I network better?
- How do I meet the right people?
- How do I maintain connections?
