# NET-005 Networking Intelligence Framework Library

## Relationship Building Framework

Purpose:
Create authentic connections.

Steps:

1. Identify people and communities
2. Start meaningful conversations
3. Discover shared interests
4. Provide value
5. Maintain connection

## Follow-Up Framework

Includes:

- remember details
- reconnect intentionally
- share helpful information
- continue relationship

## Connection Value Framework

Ask:

- What can I contribute?
- What can I learn?
- Who else could benefit?

## ADHD Application

Creates reminders and systems for maintaining relationships.
