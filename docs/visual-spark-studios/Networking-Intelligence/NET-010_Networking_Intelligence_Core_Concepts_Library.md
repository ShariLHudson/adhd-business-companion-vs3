# NET-010 Core Concepts Library

## Networking

The process of developing mutually valuable relationships.

## Relationship Capital

The value created through trust, connection, and shared opportunities.

## Community

A group connected by shared interests, goals, or purpose.

## Reciprocity

Creating relationships where value flows both ways.

## Founder Translation

Strong networks create opportunities, learning, support, and collaboration.

## ADHD Translation

Networking systems help maintain relationships when memory and follow-up are difficult.
