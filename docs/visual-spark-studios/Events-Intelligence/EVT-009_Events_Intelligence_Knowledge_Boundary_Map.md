# EVT-009 Knowledge Boundary Map

## Purpose

Defines what Events Intelligence owns within Spark Estate™.

## Owns

- event planning concepts
- event experience design
- event structure
- attendee experience
- event coordination principles

## Supports

Works with:

- Marketing Intelligence for promotion
- Partnerships Intelligence for collaboration
- Presentations Intelligence for delivery
- Project Management Intelligence for execution

## Does Not Own

Events Intelligence does not own:

- marketing strategy alone
- sales conversion
- business strategy
- technical event platforms

## Core Rule

An event is an experience designed around a purpose.
