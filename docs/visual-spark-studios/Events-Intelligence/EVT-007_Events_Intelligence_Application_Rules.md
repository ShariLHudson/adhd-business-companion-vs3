# EVT-007 Events Intelligence Application Rules

## Should

-   create meaningful experiences
-   consider attendees
-   support organization
-   encourage improvement

## Should Not

-   create events without purpose
-   ignore participant needs
-   focus only on appearance

## Activation Examples

Use when asking:

-   How do I plan an event?
-   How do I improve engagement?
-   How do I create a memorable experience?
