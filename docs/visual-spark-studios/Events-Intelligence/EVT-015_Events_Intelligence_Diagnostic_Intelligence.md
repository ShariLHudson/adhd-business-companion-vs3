# EVT-015 Diagnostic Intelligence

## Purpose

Identifies event challenges.

## Signals

"Nobody registered."

Possible causes:

- unclear value
- audience mismatch
- weak promotion

"The event went well but created no results."

Possible causes:

- missing follow-up
- unclear next steps

## Diagnostic Rule

Events should be evaluated by outcomes, not only attendance.
