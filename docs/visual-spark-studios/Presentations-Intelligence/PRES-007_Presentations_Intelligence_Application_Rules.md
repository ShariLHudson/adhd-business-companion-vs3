# PRES-007 Presentations Intelligence Application Rules

## Should

- create clarity
- support confidence
- focus on audience needs
- simplify complex ideas

## Should Not

- overload slides
- present without purpose
- prioritize information over understanding

## Activation Examples

Use when asking:

- How do I create a presentation?
- How do I explain this idea?
- How do I become a better speaker?
