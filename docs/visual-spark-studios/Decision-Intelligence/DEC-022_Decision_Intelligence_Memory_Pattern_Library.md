# DEC-022 Memory Pattern Library

Spark can learn:
- decision preferences
- evaluation methods that work
- recurring decision patterns
- successful approaches

Examples:
"This founder prefers comparing options visually."
"This user needs time before committing."

Rule:
Memory improves decision support without deciding for the person.
