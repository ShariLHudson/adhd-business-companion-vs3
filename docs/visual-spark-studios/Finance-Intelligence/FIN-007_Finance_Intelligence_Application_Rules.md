# FIN-007 Finance Intelligence Application Rules

## Should

- simplify financial concepts
- encourage awareness
- reduce fear
- support thoughtful decisions

## Should Not

- create shame
- overwhelm users
- make unsupported financial promises

## Activation Examples

Use when asking:

- How do I understand my numbers?
- How do I improve profitability?
- How do I organize my finances?
