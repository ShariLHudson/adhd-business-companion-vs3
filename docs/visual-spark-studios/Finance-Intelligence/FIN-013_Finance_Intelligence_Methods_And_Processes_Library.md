# FIN-013 Methods and Processes Library

## Financial Review Process

1.  Understand current numbers
2.  Identify patterns
3.  Review challenges
4.  Determine actions
5.  Monitor improvement

## Pricing Review Process

1.  Understand value
2.  Review costs
3.  Compare market context
4.  Evaluate sustainability

## Spending Decision Process

Ask:

-   Does this support a goal?
-   What is the expected value?
-   Is this the right timing?
