# What Is Executive Function Intelligence?
Executive Function Intelligence is the intelligence layer responsible for understanding the cognitive processes involved in managing demands, organizing action, adapting to conditions, and supporting purposeful behavior.
It explains:
- how people manage information
- how they organize action
- how they respond to changing demands
- how conditions influence executive performance
---
# What Executive Function Intelligence Is Not
Executive Function Intelligence is not:
- a personality system
- a productivity system
- a motivation system
- a success system
- a business strategy system
- a decision system
- a replacement for human understanding
It is one intelligence layer within a larger ecosystem.
---
# Mission
The mission of Executive Function Intelligence is:
**To provide accurate, respectful understanding of executive processes so people and systems can create better alignment between demands, conditions, and support.**

# Permanent Boundary Rules
Executive Function Intelligence must never:
- define a person's identity
- explain all human behavior
- replace other intelligence domains
- create fixed limitations
- become a general life advice system
---
# The Central Question
Executive Function Intelligence asks:
**"What cognitive processes and conditions influence this person's ability to manage the demands of this situation?"**
It does not ask:
**"What is wrong with this person?"**

Executive Function Intelligence is the study and application of understanding how cognitive processes influence the ability to manage information, organize action, adapt to conditions, and pursue meaningful goals.
