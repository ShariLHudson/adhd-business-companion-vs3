# EXF-009 Working Memory Intelligence
Defines Working Memory as an executive function domain and establishes how Executive Function Intelligence understands the process of temporarily holding, using, and manipulating information during meaningful action.

# Purpose
People continuously interact with information.
They remember instructions while completing tasks.
They hold ideas while making decisions.
They connect previous information with current situations.
They maintain context while moving through activities.
Working Memory Intelligence exists to understand the executive processes involved in temporarily holding and using information needed for purposeful action.
It answers the question:
**"How does a person manage information that must remain available while they think and act?"**

# Working Memory Defined
Working Memory is the executive function process of temporarily maintaining and manipulating information while completing meaningful activity.
It supports activities such as:
- following multi-step instructions
- keeping relevant information available
- connecting ideas
- managing immediate priorities
- holding context during action
Working Memory is limited and influenced by current conditions.
---
# Guiding Principles
## Working Memory Is Limited
People have a limited capacity for actively holding information at one time.
When demands exceed available capacity, errors and difficulty may increase.
---
## External Support Strengthens Function
People naturally use external systems to reduce Working Memory demands.
Examples include:
- notes
- reminders
- written instructions
- visual references
- checklists
External support is an adaptation, not a weakness.
---
## Context Influences Working Memory
Working Memory demands change depending on:
- familiarity
- complexity
- interruptions
- stress
- competing information
- environmental conditions
The same task may require different levels of Working Memory in different situations.
---
## Working Memory Supports Continuity
Maintaining context allows people to continue meaningful activity without repeatedly reconstructing information.
---
## Simplicity Preserves Working Memory
Reducing unnecessary complexity protects available cognitive resources.
Clear systems allow Working Memory to be used for what matters most.
---
## Working Memory Improves Through Understanding
Familiarity, experience, and meaningful organization can reduce the amount of information that must be actively managed.
---
# Working Memory Components
Executive Function Intelligence may consider:
## Holding Information
Keeping relevant information available during action.
---
## Manipulating Information
Using information to compare, connect, or transform ideas.
---
## Maintaining Context
Remembering what is happening and why it matters.
---
## Updating Information
Adjusting understanding as new information becomes available.
---
## Filtering Information
Determining what information requires active attention.
---
# Working Memory and Other Executive Domains
Working Memory interacts with multiple executive processes.
## Planning
Planning often requires holding future steps and outcomes in mind.
---
## Organization
Organization reduces the amount of information that must be remembered internally.
---
## Attention Management
Attention determines what information receives cognitive resources.
---
## Prioritization
Prioritization helps determine which information matters most.
---
## Self-Monitoring
Self-monitoring uses available information to evaluate progress.
---
Working Memory Intelligence recognizes these relationships without redefining other executive domains.

# Relationship With Knowledge Management
Working Memory and Knowledge Management support each other but serve different purposes.
Working Memory concerns temporary active information management.
Knowledge Management concerns preserved knowledge and information relationships.
Externalizing information can reduce Working Memory demands while strengthening knowledge access.
