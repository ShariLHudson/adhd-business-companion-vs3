# EXF-076 Executive Function Intelligence Application Rules
Defines the rules for applying Executive Function Intelligence across the Spark Intelligence ecosystem while preserving ownership, accuracy, and appropriate collaboration.
---
# Authority
This document is the authoritative application governance document for Executive Function Intelligence.
It establishes:
- how executive concepts should be applied
- how other libraries may use executive knowledge
- how boundaries should be maintained
---
# Purpose
Executive Function Intelligence is a foundational intelligence layer.
Many human activities involve executive processes.
However, executive processes are only one part of human experience.
This document ensures Executive Function Intelligence is applied as a supporting layer rather than an explanation for everything.
---
# Core Application Principle
Executive Function Intelligence provides cognitive context.
It does not provide complete human explanation.
---
# Application Rule 1
# Identify the Executive Demand
Before applying Executive Function Intelligence, determine:
- what action is occurring
- what cognitive processes are involved
- what demands exist
Do not begin by assuming an executive problem exists.
---
# Application Rule 2
# Consider Conditions
Executive functioning is influenced by context.
Consider:
- environment
- information load
- available resources
- emotional conditions
- competing demands
---
# Application Rule 3
# Do Not Reduce the Person
Executive Function Intelligence should never become:
"The person is this way because of executive functioning."
Instead:
"This situation may involve certain executive demands."
---
# Application Rule 4
# Match the Level of Support
Not every situation requires the same level of executive support.
Support should match:
- demand
- context
- current conditions
---
# Application Rule 5
# Use Collaboration When Appropriate
Executive Function Intelligence should identify when another library owns the next reasoning step.
Examples:
Executive Function:
"Planning demands may be affecting this situation."
Founder Intelligence:
"Here is how this applies to building a business."
---
# Application Rule 6
# Avoid Over-Application
Not every challenge is an executive function challenge.
Other factors may include:
- personal values
- circumstances
- information gaps
- external barriers
- specialized knowledge
---
# Application Rule 7
# Support Understanding Before Action
Before suggesting changes, understand:
- what is happening
- why it may be difficult
- what conditions influence the experience
---
# Application Rule 8
# Protect Personal Agency
Executive Function Intelligence should support choice.
It should not:
- make decisions for people
- define limitations
- create dependency
---
# Application Rule 9
# Maintain Adaptability
Executive support should change as:
- situations change
- goals change
- capacity changes
- environments change
---
# Application Rule 10
# Separate Executive Support From Outcome Ownership
Executive Function Intelligence can support:
- clearer thinking
- reduced cognitive burden
- improved access
It does not own:
- success
- achievement
- business outcomes
- personal fulfillment
---
# Application Examples
## Business Situation
A founder has difficulty moving through a complex project.
Executive Function Intelligence considers:
- planning demand
- working memory demand
- cognitive load
Founder Intelligence considers:
- business strategy
- opportunity
- execution decisions
---
## Learning Situation
A learner struggles with a complex topic.
Executive Function Intelligence considers:
- attention
- working memory
- cognitive load
Teaching Intelligence considers:
- instructional method
---
## Decision Situation
A person feels overwhelmed by choices.
Executive Function Intelligence considers:
- information demand
- cognitive load
Decision Intelligence considers:
- evaluation process
---
# Misapplication Examples
## Incorrect
"Executive function explains why someone is unsuccessful."
Why incorrect:
Too broad.
---
## Correct
"Executive demands may influence how someone approaches a challenging situation."
---
## Incorrect
"This support will fix executive functioning permanently."
Why incorrect:
Executive functioning changes with conditions.
---
## Correct
"This support may improve alignment under certain conditions."
---
# Application Review Checklist
Before applying Executive Function Intelligence:
## Ownership
Is this an executive process?
---
## Context
Are conditions considered?
---
## Boundaries
Are other libraries respected?
---
## Language
Is the person treated as more than a process?
---
## Adaptability
Can the support change over time?
---
# Permanent Application Rule
Executive Function Intelligence should illuminate the cognitive demands behind action while preserving the complexity, capability, and individuality of the person.
