# EXF-063 Cognitive Load Intelligence
## Authority
Defines Cognitive Load as an Executive Function concept.
---
# Purpose
Cognitive Load Intelligence explains the amount of mental effort required to process information, manage demands, and complete activities.
---
# Scope
Owns:
- mental demand
- information complexity
- processing burden
Does not own:
- stress management
- emotional health
- productivity optimization
---
# Cognitive Load Components
## Information Volume
Amount of information being processed.
## Complexity
Difficulty of understanding relationships.
## Processing Requirements
Mental effort required.
## Competing Demands
Multiple simultaneous requirements.
---
# Cognitive Load Principles
High cognitive load may affect:
- working memory
- planning
- attention
- decision-making
Reducing unnecessary cognitive load can improve access.
---
# Relationship With Executive Capacity
Cognitive Load describes demand.
Executive Capacity describes available resources.
Together they explain demand-resource alignment.
---
# Permanent Ownership Rule
Cognitive Load Intelligence owns understanding mental demand.
