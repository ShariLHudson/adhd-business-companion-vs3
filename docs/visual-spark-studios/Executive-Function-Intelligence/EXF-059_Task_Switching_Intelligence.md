# EXF-059 Task Switching Intelligence
## Authority
Defines Task Switching as an Executive Function concept.
---
# Purpose
Task Switching Intelligence explains the executive process involved when moving attention and resources between activities.
---
# Scope
Owns:
- shifting attention
- changing task focus
- managing switching demands
Does not own:
- multitasking strategies
- productivity workflows
---
# Task Switching Components
## Preparation
Understanding what is required before shifting.
## Disengagement
Releasing attention from the current task.
## Reorientation
Directing attention toward the next task.
## Context Restoration
Recovering information needed to continue.
---
# Relationship With Transitions
Transitions describe movement between states.
Task Switching describes movement between tasks.
They collaborate but are not identical.
---
# Permanent Ownership Rule
Task Switching Intelligence owns understanding cognitive switching demands.
