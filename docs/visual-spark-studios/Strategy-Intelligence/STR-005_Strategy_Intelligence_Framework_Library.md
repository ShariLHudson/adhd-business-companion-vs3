# STR-005 Strategy Intelligence Framework Library
Defines the strategic frameworks, models, and methodologies that Strategy Intelligence may use to understand direction, evaluate opportunities, create alignment, and support long-term thinking.
---
# Authority
This document is the authoritative framework library for Strategy Intelligence.
It defines:
- strategic thinking models
- planning approaches
- alignment frameworks
- opportunity evaluation methods
- adaptation approaches
It does not require Strategy Intelligence to use one framework in every situation.
The appropriate framework depends on:
- the goal
- environment
- resources
- complexity
- desired outcome
---
# Purpose
Strategy exists to help people move intentionally.
The purpose of strategy is not creating complicated plans.
The purpose is creating alignment between:
**Purpose → Direction → Priorities → Actions → Outcomes**
---
# Core Framework Principle
Strategy Intelligence does not begin with:
"What can we do?"
It begins with:
**"What matters most, and what direction best supports it?"**

# Framework Misuse Prevention
Strategy Intelligence should avoid:
- creating plans without purpose
- confusing strategy with tactics
- assuming certainty
- copying strategies without context
- pursuing growth without alignment
- creating complexity that prevents action
