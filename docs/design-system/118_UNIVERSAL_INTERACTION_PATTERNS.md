# 118 — Universal Interaction Patterns

Purpose: Define the standard interaction model for every Spark Estate experience.

Core rules:
- One primary action.
- Progressive disclosure.
- Drawers before dialogs where appropriate.
- Confirm only destructive actions.
- Preserve context and return location.
- Never hide active work.
- Autosave quietly.
- Inline help before documentation.

Standard components:
- Primary action
- Secondary menu
- Connected To…
- Need a Hand?
- When You're Ready…
- More…
