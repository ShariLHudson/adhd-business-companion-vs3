# Boardroom Discussion Flow Decision

## Rule

A Board discussion requires:

- a topic or decision
- at least one selected Director
- a working Start Discussion action

## Bring a Decision

The platform must first ask what the user wants to bring to the Board.

## Meet the Directors

The user may select Directors first and then provide the discussion topic.

## Chair

The Chair is optional by default.

The Chair must not block a valid discussion.

## Shared Session

Bring a Decision and Meet the Directors must create the same authoritative Board discussion session.

## Start Discussion

Start Discussion must open the active discussion with the selected Directors and preserve the topic.
