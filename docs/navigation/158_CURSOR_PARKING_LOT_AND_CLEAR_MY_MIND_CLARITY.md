# 158_CURSOR_PARKING_LOT_AND_CLEAR_MY_MIND_CLARITY

## Goal

Users should immediately understand which experience to use.

---

# Clear My Mind

### Purpose

**Empty everything that's competing for your attention.**

Use Clear My Mind when you have:

- lots of thoughts
- overwhelm
- a brain full of ideas
- worries
- tasks
- reminders
- things you don't want to forget

Shari helps separate, organize and decide what happens next.

Example text:

> Feeling overwhelmed? Put everything on your mind here. I'll help sort it into manageable next steps.

CTA:
**Start Emptying My Mind**

---

# Park It

### Purpose

**Save one thing for later so you can stop thinking about it right now.**

Use Park It when you have:

- one task
- one idea
- one worry
- one reminder
- one follow-up

that you don't want to deal with yet.

Example text:

> Not ready to deal with this yet? Park it here and I'll keep it safe until you're ready.

CTA:
**Park This Item**

---

# Parking Lot

The Parking Lot is not just an action.

It is a destination where users can review everything they parked.

Each parked item should support:

- Move to Today's Plan
- Create Reminder
- Add to Project
- Edit
- Delete
- Leave Parked

Display:

- title
- parked date
- category
- status
- search/filter

---

# Relationship

Clear My Mind
→ capture many thoughts
→ organize them
→ some may become reminders
→ some may become projects
→ some may go to the Parking Lot

Park It
→ capture one item
→ place it directly into the Parking Lot

---

# UI

Show a short explanation under both cards.

Do not require users to guess the difference.

Never use the same description for both experiences.
