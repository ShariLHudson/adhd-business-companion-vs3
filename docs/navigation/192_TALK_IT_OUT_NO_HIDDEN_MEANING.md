# 192 — Talk It Out: No Hidden Meaning & Direct Repair

## Principle

Do not invent a quieter / deeper issue when the user stated a clear practical question. Corrections override interpretation.

## Pipeline

1. Literal meaning  
2. **Direct correction** (immediate repair)  
3. Explicit help  
4. CRCI  
5. RCI (evidence-gated)  
6. CI → Grounded Ack → CQRI → send  

## Thinking Map (internal)

- `literalTopic`
- `rejectedInterpretations`
- `userCorrections`
- `interpretationEvidenceCount`
- `interpretationAllowed` (≥ 2 grounded signals)
