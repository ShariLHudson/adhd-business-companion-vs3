# Create and Peaceful Places Decision

## Create

Create must have:
- one reliable vertical scroll owner
- full access to all content
- no clipped controls
- no blocked items
- complete mouse, keyboard, and touch interaction

## Peaceful Places

Peaceful Places contains:
- the approved woodland background
- one scrollable music dropdown
- visible playback controls

It does not contain:
- guided activities
- journaling
- breathing exercises
- multiple experience cards
- automatic audio

## Sound Rule

Sound is always opt-in.

No sound starts until the user deliberately starts it.

The user must always be able to:
- pause
- stop
- mute
- turn sound off
- stop all sound
