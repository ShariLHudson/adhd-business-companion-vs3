# Plan My Day Functional Decision

## Approved Purpose

Plan My Day is not only a task-entry list.

It must transform the user's task list into a realistic plan for today.

## Required Progression

1. capture tasks
2. separate and clarify tasks
3. consider calendar, time, energy, and motivation
4. identify priorities
5. estimate effort
6. fit the plan to reality
7. create a useful sequence
8. provide one concrete first step
9. save and resume
10. connect to Adapt My Day

## “I'm Stuck”

This action provides contextual planning help and must not route automatically to generic calming content.
