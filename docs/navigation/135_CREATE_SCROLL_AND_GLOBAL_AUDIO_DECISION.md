# Create Scroll and Global Audio Decision

## Create

The Create destination must:
- have one reliable vertical scroll owner
- keep every control reachable
- keep every control clickable
- avoid invisible pointer blockers and click-through

## Sound

All Spark Estate sound is opt-in.

No sound starts without explicit user action.

A global visible audio control must provide:
- Play
- Pause
- Stop
- Mute
- Volume
- Stop All Sound

## Default

Autoplay is disabled globally.
