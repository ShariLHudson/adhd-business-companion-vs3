# Shared My Day Windows — Navigation Decision

## Approved Structure

### Shared Window 1

Plan My Day / Adapt My Day

Contains:

- Plan My Day
- Adapt My Day
- one shared How Do I…
- explanations for both items
- one scrollable window

### Shared Window 2

Reminders / Rhythms

Contains:

- Reminders
- Rhythms
- one shared How Do I…
- explanations for both items
- clear Reminder versus Rhythm distinction
- one scrollable window

## Core Difference

Reminder:

One specific thing to remember.

Rhythm:

Recurring support for something the user wants to keep returning to.
