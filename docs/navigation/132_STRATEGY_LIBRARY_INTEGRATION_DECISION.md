# Strategy Library Integration Decision

## Approved Role

The Strategy Library is the platform's shared strategy action layer.

It owns:

- strategy records
- strategy workflows
- personalization
- implementation
- progress
- lifecycle

## Connected Capabilities

- Chamber contributes expertise
- Board contributes review and challenge
- Visual Thinking contributes representation
- Projects contributes ongoing execution
- Plan My Day contributes today's action
- Reminders contribute one-time follow-up
- Rhythms contribute recurring support
- Shari orchestrates and communicates

## Core Constraint

Connected capabilities contribute to the strategy.

They do not replace Strategy Library ownership.
