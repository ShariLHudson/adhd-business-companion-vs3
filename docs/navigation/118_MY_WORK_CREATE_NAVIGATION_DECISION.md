# My Work Create Navigation Decision

## Approved Change

Add:

**Welcome Home → My Work → Create**

Create is a first-class destination for intentionally making something new.

## Create Owns

- starting a new creation
- understanding what the user wants to make
- routing to the correct creation capability
- browsing supported creation types
- intentionally resuming valid saved creations

## Create Does Not Replace

- Projects
- Strategy Library
- the global intent/workflow gate
- capability-specific creation flows

## Core Distinction

Create makes an output.

Projects organize ongoing work.
