## Visit ID format

```
YYYY-MM-DD #N — [slot] — [context optional]
```

Examples:

- `2026-06-26 #1 — morning`
- `2026-06-26 #3 — late-afternoon — after productive session`
- `2026-06-26 #5 — late-night — overwhelmed`

## Rules

- **Honest over kind** — this is for craftsmanship, not morale  
- **One tiny improvement** per visit — resist the redesign urge  
- **Rewrite Shari's lines** in Q4 when they fail — paste improved copy in log  
- **Software moments** (Q3) are gold — be specific: *what you saw, what you felt*

## Sprint close checklist

- [ ] ≥35 visit records logged  
- [ ] All five time-of-day slots covered multiple days  
- [ ] All five context visits completed at least once  
- [ ] Weekly synthesis written at bottom of `sprint-zero-log.md`  
- [ ] Fix queue ≤3 items approved for craftsmanship PRs
