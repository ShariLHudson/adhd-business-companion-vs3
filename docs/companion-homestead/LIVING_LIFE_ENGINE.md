# Part I ‚Äî The Living Life Principle‚Ñ¢

## Core question (every entry, every return)

> **What has naturally changed since the last time this person was here?**

| Rule | Meaning |
|------|---------|
| **Never force change** | Absence of novelty is valid ‚Äî sameness is sometimes the gift |
| **Never change for novelty** | No rotation "because it's Tuesday" |
| **Only change because life moves** | Time, weather, season, relationship, chapter, rhythm, movement between rooms |

## Recognition test

| Fail | Pass |
|------|------|
| "They reskinned the app." | "The house is the same ‚Äî something small shifted." |
| "New UI again." | "Oh ‚Äî cookies. She must have been baking." |
| "Why is everything different?" | "Rain today. Of course the lamp is on." |

## The One Constant‚Ñ¢

While details gently evolve:

- **Architecture** never changes ‚Äî same rooms, same materials, same Iowa honesty
- **Personality** never changes ‚Äî Shari is Shari; Kinsey is Kinsey; the homestead is Shari's
- **Emotional promise** never changes ‚Äî *you are not alone; you are welcomed; you may rest*
- The guest always feels: **"I came back to a place I love."**

# Part III ‚Äî Daily Rhythm System‚Ñ¢

## Principle

Not every visit begins the same way. Rhythm is **habit with variation** ‚Äî like a real home where sometimes you're met at the door and sometimes you hear *"come on in"* from the kitchen.

## Rhythm dimensions

| Dimension | Variants |
|-----------|----------|
| **Who leads** | Room first ¬∑ Shari first ¬∑ Silence first ¬∑ Guest first (typed early) |
| **Shari location** | Living Room host ¬∑ Nearby off-camera ¬∑ Garden (implied) ¬∑ Reading (implied) ¬∑ Not present (voice) |
| **Opening beat** | Settle only ¬∑ Greet only ¬∑ Greet + sit ¬∑ Reality question ¬∑ Observation only |
| **Sound entry** | Silence ¬∑ Birds ¬∑ Rain ¬∑ Fire crackle ¬∑ Floorboard (future audio) |
| **Light entry** | Morning window ¬∑ Lamp pool ¬∑ Fire glow ¬∑ Gray rain day |

## Rhythm selection (not random)

Rhythm variant = `f(time band, relationship tenure, life chapter, last rhythm, today's capacity, room return context)`

**Hard rules:**

- **Low capacity / flooded** ‚Üí silence-first or room-first; never interrogation rhythm
- **First week** ‚Üí host-present rhythm; shorter silence
- **Year five** ‚Üí silence permitted; observation replaces questions on ordinary days
- **Same-day return** (< 4 hours) ‚Üí compress; no full arrival re-run
- **Never same rhythm 4 consecutive visits** unless life chapter locks it (e.g., recovery)

## Rhythm catalog (12 archetypes)

| ID | Name | When |
|----|------|------|
| R01 | **Threshold** | First visits, long absence ‚Äî settle ‚Üí greet ‚Üí sit |
| R02 | **Kitchen voice** | Morning ordinary ‚Äî *"Come on in"* off-camera |
| R03 | **Already here** | Established ‚Äî Shari in frame, one line |
| R04 | **Room greets** | Visual change carries entry ‚Äî no copy 2s |
| R05 | **Rain day** | Weather-led ‚Äî lamp/fire before words |
| R06 | **Quiet return** | Same-day ‚Äî no question |
| R07 | **Chapter hush** | Grief, recovery ‚Äî silence beats |
| R08 | **Celebration wait** | Win context ‚Äî room prepared before boast |
| R09 | **Garden implied** | Spring morning ‚Äî Shari "outside," voice only |
| R10 | **Reading nook drift** | Evening ‚Äî softer, fewer questions |
| R11 | **Weekend late** | No planning bias ‚Äî rest rhythm |
| R12 | **Return from work** | Room return layer ‚Äî continuity not re-arrival |

Maps to `First Production Experience‚Ñ¢` beats ‚Äî rhythm selects **depth**, not new patterns.

# Part IV ‚Äî Relationship Evolution Model‚Ñ¢

## Principle

People who know each other talk differently over time. Not scripts ‚Äî **register shift**.

## Tenure bands

| Band | Approximate | Conversational character |
|------|-------------|--------------------------|
| **T0 ‚Äî Stranger** | First 1‚Äì3 visits | Host formality, full welcomes, gentle questions |
| **T1 ‚Äî Guest** | Week 1 | Warm, still explaining nothing ‚Äî offering presence |
| **T2 ‚Äî Regular** | 2‚Äì8 weeks | Shorter greetings, familiar objects, "there you are" |
| **T3 ‚Äî Familiar** | 2‚Äì6 months | Observations over questions; humor permitted |
| **T4 ‚Äî Trusted** | 6 months ‚Äì 2 years | Silence comfortable; direct honesty |
| **T5 ‚Äî Home** | 2‚Äì5 years | Minimal ceremony; depth on cue |
| **T6 ‚Äî Life** | 5+ years | Shared history in the air; rare perfect lines |

## Evolution axes (independent)

| Axis | Low tenure | High tenure |
|------|------------|-------------|
| **Question frequency** | Higher | Lower ‚Äî observations replace |
| **Explanation** | More context | Assumes shared map |
| **Silence permission** | Low | High |
| **Vulnerability match** | Gentle | Can be blunt-kind |
| **Celebration dwell** | Brief | Longer ‚Äî earned joy |
| **Absence greeting** | Recovery warmth | Ordinary ‚Äî no guilt |

## Never

- Streak language ¬∑ absence shame ¬∑ "I noticed you haven't‚Ä¶"
- Personality drift ¬∑ new catchphrases without founder approval
- Fake intimacy before trust earned (Trust Architecture‚Ñ¢)

## Conversation Intelligence‚Ñ¢ (greeting grammar)

| Move | Example | When |
|------|---------|------|
| **Welcome** | *"Come on in."* | Universal |
| **Observation** | *"Rain's been at the windows all morning."* | Weather/time led |
| **Recognition** | *"There you are."* | T2+ |
| **Continuity** | *"Still thinking about yesterday's idea?"* | Carry context only |
| **Permission** | *"No rush today."* | Low capacity |
| **Silence** | (none) | T4+, grief, late night |

**Anti-AI phrases blocklist:** delighted ¬∑ absolutely ¬∑ great question ¬∑ as an AI ¬∑ I'd be happy to ¬∑ leverage ¬∑ journey (unless user's word)

**Anti-repetition:** no exact greeting line within 14 visits; no same opening observation within 7 rain days.

# Part V ‚Äî Life Continuity Model‚Ñ¢

## Principle

The Homestead remembers life chapters **naturally** ‚Äî never surveillance, never *"I noticed you seem sad."*

Continuity is expressed through **preparation, tone, and object truth** ‚Äî not narration about memory.

## Chapter types (inferred, never labeled in UI)

| Chapter signal | Environmental bias | Conversational bias | Duration |
|----------------|-------------------|---------------------|----------|
| **Launch week** | Creative studio warmth; mug; later hours lamp | Match energy; don't pivot to admin | Days‚Äì2 weeks |
| **Recovery** | Blanket, low light, fewer objects | Minimal questions; Window Seat bias | Until signal fades |

## Environmental Truth‚Ñ¢ (implementation)

Nothing moves because animation is available. Every effect traces to a cause:

| Cause | Effect |
|-------|--------|
| Window open | Curtains stir |
| Summer breeze | Candle flicker, curtains |
| Fresh coffee / tea | Steam |
| Morning sun | Warm daylight |
| Wind outside | Foliage / branches sway |
| Rain outside | Rain motion, lamp warmth ‚Äî no harsh sun |
| Winter Iowa | Snow when warranted; quiet wildlife |
| Summer Iowa | Lush foliage, open-window breeze |

Module: `lib/environmentalTruth/` ‚Äî runs after Living Change Engine, before render (`applyEnvironmentalTruth` in `evaluateCompanionEnvironmentIntelligence`).

Narrative grammar is internal (`because` lines): *"Of course the curtains are moving ‚Äî the window is open."* Never shown as UI chrome.

## Companion Communication Anchor‚Ñ¢ (implementation)

Living Change Engine‚Ñ¢ and arrival choreography **must not**:

- hide chat behind menus or overlays  
- remove mic during room transitions  
- let decorative layers block input (`pointer-events: none` on life layers; anchor stays `pointer-events: auto`)  
- auto-focus during arrival pause beats (`quiet` mode only)

Every room shell renders `CompanionCommunicationAnchor`. Arrival uses `quiet` mode during settle/greet/sit/walk and `full` mode during reality/invite/staying.
