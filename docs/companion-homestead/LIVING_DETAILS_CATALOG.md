```typescript
type LivingDetailCandidate = {
  id: string;           // e.g. "D001"
  category: string;
  signalRequirements: LivingSignal[];  // min one from Layers 1–7
  roomAllowlist: CompanionPlaceId[];
  cooldownVisits: number;
  cooldownDays: number;

};
```

**Selection:** rank by signal strength → apply cooldowns → cap at 2 → integrity veto → Brain may zero.
