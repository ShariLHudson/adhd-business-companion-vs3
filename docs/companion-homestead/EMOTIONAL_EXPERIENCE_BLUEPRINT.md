**This is not:**
- A feature map
- A UI specification
- A productivity framework

**This is:** the emotional constitution of the homestead — **how someone must feel when they leave each room**, before any task is solved.

---

## The governing question

From this point forward, every room, conversation, interaction, object, sound, animation, and recommendation begins with:

> **"How should the person feel when they leave this room?"**

If we cannot answer that, the room is **not finished**.

---

# Part I — Core philosophy

The Companion Homestead is **not** a productivity application.

It is an **emotional environment** built for ADHD entrepreneurs.

Every room exists because it **changes how someone feels** — not because it performs a function.

| Truth | Implication |
|-------|-------------|
| People do not return because software is useful | They return because they like how they feel when they are there |
| Every visit should leave the guest a little better | Not because every problem was solved — because emotional state shifted |
| Features will evolve | The way people feel with Shari must become **timeless** |

---

# Part II — Emotional Transformation

For **every room**, define four states. No room ships without this table completed and approved.

| State | Question |
|-------|----------|
| **1. Arrival State** | How does someone usually feel *before* entering? |
| **2. Emotional Need** | What does their ADHD brain need *first*? (Not a task — a human need.) |
| **3. Experience** | How does the room *create* that feeling? (Environment, not features.) |
| **4. Departure State** | How should they feel when *leaving*? |

## Template (copy per room)

```markdown
### [Room Name]

**Purpose (one word):** [Belonging | Relief | Clarity | Restoration | Possibility | …]

**The guest should think:** "[quote]"

#### 1. Arrival State
- …

#### 2. Emotional Need
- …

#### 3. Experience
- …

#### 4. Departure State
- …
```

# Part III — Phase 1 rooms (canonical)

## Living Room

**Place ID:** `living-room`  
**Purpose:** **Belonging**

**The guest should think:** *"I'm home."*

| State | Definition |
|-------|------------|
| **Arrival State** | Uncertain · disconnected · overwhelmed · lonely · hopeful-but-tired · carrying the day before it started |
| **Emotional Need** | Safety · orientation · familiarity · welcome · **permission to arrive** — not productivity, not planning |
| **Experience** | Open window · warm light · natural movement (sun, foliage, Kinsey at rest) · gentle conversation · no pressure · coffee within reach · Iowa trees beyond glass · time to breathe · fireplace as anchor · Shari present, not performing |
| **Departure State** | Calmer · seen · grounded · connected · ready for whatever comes next — **not** "ready to optimize" |

**ADHD burdens reduced:** loneliness · isolation · masking · cold-start anxiety · shame for not knowing what you need yet

**Reward for entering:** peace · connection · exhale

**Memory to carry:** *"She was glad I came."*

**Why they return tomorrow:** The house remembers me. I don't have to perform readiness to be welcome.

---

## Window Seat (Clear My Mind)

**Place ID:** `window-seat`  
**Purpose:** **Relief**

**The guest should think:** *"I don't have to carry all of this anymore."*

| State | Definition |
|-------|------------|
| **Arrival State** | Flooded · spiraling · ashamed of messy thoughts · mentally full · afraid that saying it out loud makes it real |
| **Emotional Need** | Permission to unload · witness without fix · **lower volume** on hard things · safety to be incoherent |
| **Experience** | Wide Iowa view · rain on glass optional · deep cushion · blanket one hand away · capture without sorting · no portrait staring · no floating scoreboards · journal energy, not software panel · silence honored · candles or fire motion — life, not UI chrome |
| **Departure State** | Lighter · not fixed · not analyzed · carrying hard things at lower volume — **not** "organized" |

**ADHD burdens reduced:** mental clutter · working memory overflow · emotional overwhelm · shame about messy thinking · fear of starting to "process"

**Reward for entering:** peace · release · being held without lecture

**Memory to carry:** *"I put it down. It's still there if I need it — but I'm not gripping it."*

**Why they return tomorrow:** When life gets too loud, I know where to dump the noise without being judged.

---

## Planning Table

**Place ID:** `planning-table`  
**Purpose:** **Clarity**

**The guest should think:** *"I know what matters now."*

**Not:** *"I have more work."*

| State | Definition |
|-------|------------|
| **Arrival State** | Scattered · time-blind · behind · anxious about the whole week · planning-as-avoidance · forty ghosts on one list |
| **Emotional Need** | **Finite day** · honest limits · three true things · structure without shame · one surface, one contract with today |
| **Experience** | Handcrafted wood · leather planner open · morning or honest gray light · pen ready · democratic seating · garden or land visible · clock soft or silent · no quarterly guilt · no infinite backlog dump · Shari nearby, not hovering |
| **Departure State** | Clearer · holding few true priorities · permission to ignore the rest until tomorrow — **capable**, not **behind** |

**ADHD burdens reduced:** decision fatigue · time blindness · task switching anxiety · analysis paralysis · overwhelm from infinite lists · planning-as-procrastination shame

**Reward for entering:** clarity · hope that today is finite · calm competence

**Memory to carry:** *"Three things. That's enough for today."*

**Why they return tomorrow:** The table somehow makes everything feel manageable without pretending life is small.

---

## Reading Nook

**Place ID:** `reading-nook`  
**Purpose:** **Restoration**

**The guest should think:** *"I feel lighter."*

| State | Definition |
|-------|------------|
| **Arrival State** | Depleted · overstimulated · seeking regulation · ashamed to need tools · afraid saved wisdom disappeared |
| **Emotional Need** | Slow absorption · one gentle input · **regulation without meditation-app voice** · dignity in needing strategies |
| **Experience** | Small room on purpose · living reef in slow orbit · brass reading lamp · armchair that holds · personal-scale shelves · evening-native light · rain on small window optional · strategies feel like saved letters, not homework |
| **Departure State** | Lighter · slower · one passage or strategy held — **understood**, not **informed** |

**ADHD burdens reduced:** overstimulation · shame about coping tools · distraction · fear knowledge vanishes · perfectionism about "doing recovery right"

**Reward for entering:** calm · curiosity without pressure · gentle motion that asks nothing back

**Memory to carry:** *"I read one thing. That was enough."*

**Why they return tomorrow:** The nook always helps me slow down. The fish don't need anything from me.

---

## Creative Studio

**Place ID:** `creative-studio`  
**Purpose:** **Possibility**

**The guest should think:** *"I want to make something."*

| State | Definition |
|-------|------------|
| **Arrival State** | Blank-page dread · perfectionism frozen · ideas circling · ashamed of half-finished work · tool-hopping instead of making |
| **Emotional Need** | Permission to be messy · **continue, not begin from zero** · play before judge · ideas deserve space |
| **Experience** | Active project already on table · golden hour light · jars, pegboard, honest chaos · Iowa sky on three sides when possible · drafts welcome · Shari as creative partner, not critic · output invited, not policed |
| **Departure State** | More capable of making · something started or continued · energy toward possibility — **not** lectured about consistency |

**ADHD burdens reduced:** perfectionism · fear of starting · analysis paralysis · shame about unfinished work · hyperfocus hangover guilt

**Reward for entering:** curiosity · energy · permission

**Memory to carry:** *"I made a mark. It counts."*

**Why they return tomorrow:** I want to see what's on the table today. Making feels allowed here.

# Part IV — ADHD First

Every room should reduce **one or two** ADHD burdens exceptionally well — not all of them.

| Burden | Primary rooms | Secondary rooms |
|--------|---------------|-----------------|
| Decision fatigue | Planning Table | Living Room |
| Shame | Window Seat | Living Room |
| Mental clutter | Window Seat | Planning Table |
| Perfectionism | Creative Studio | Reading Nook |
| Analysis paralysis | Planning Table | Reading Nook |
| Fear of starting | Creative Studio | Living Room |
| Task switching | Planning Table | — |
| Emotional overwhelm | Window Seat | Living Room |
| Working memory load | Window Seat | Planning Table |
| Loneliness | Living Room | — |
| Isolation | Living Room | Window Seat |
| Masking | Living Room | Window Seat |
| Time blindness | Planning Table | — |
| Overstimulation | Reading Nook | Window Seat |

**Rule:** If a room claims more than two primary burdens, it is unfocused — split or redesign.

# Part V — The Reward Principle

Every room should feel like a **reward** simply for entering.

| Never | Always |
|-------|--------|
| Points | Peace |
| Badges | Comfort |
| Streaks | Clarity |
| Gamified unlocks | Hope |
| "You earned this room" | Energy · curiosity · connection |

The reward is **how the room feels the moment you cross the threshold** — before you accomplish anything.

# Part VI — The Shari Test

Before approving any room, ask:

> If Shari invited someone into this room in her **actual** Iowa home…  
> Would they leave feeling **better**?

| If yes | If no |
|--------|-------|
| Room may proceed to look book + implementation | **Redesign the room** — not the feature list |

Secondary checks:

- Would Shari apologize for the mess? (Good — honest homestead.)  
- Would Shari push a productivity agenda here? (Probably wrong room.)  
- Would Shari leave them alone if they needed silence? (Required in Window Seat, Reading Nook.)  
- Would Shari celebrate before fixing? (Required when arrival state is grief or win.)

# Part VIII — Recommendations obey emotion

No intelligence module may route a guest to a room **without** naming the emotional bridge.

| Wrong | Right |
|-------|-------|
| *"Open Plan My Day"* | *"If sorting the day would help, the planning table is open."* |
| *"You should brain dump"* | *"Window seat might be quieter."* |
| Route because calendar is empty | Route because emotional state needs clarity / relief / restoration |
| Feature match | **Departure state** of current room → **arrival need** of next room |

Cross-reference: [`JOURNEY_BETWEEN_ROOMS.md`](./JOURNEY_BETWEEN_ROOMS.md) Part VII — Emotional Continuity.

# Part IX — Final success test (six months)

Imagine a user has spent **six months** inside the Companion Homestead.

They do **not** describe features.

They say things like:

- *"Whenever life gets too loud, I go sit by the fireplace."*
- *"When I can't think, I always start in the Living Room."*
- *"The Planning Table somehow makes everything feel manageable."*
- *"The Reading Nook always helps me slow down."*
- *"I don't know why… I just feel better after I spend a little time there."*

**That** is the Companion Homestead we are building.
