# Conversation Engine Release Checklist

Verify:

- one active conversation state owner
- one open-question owner
- one progression decision
- one final response owner
- no duplicate visible messages
- no repeated question after a valid answer
- topic changes work
- interruptions work
- clarification is focused
- generic fallback is justified
- New Chat isolation remains intact
- specialist routing preserves context
- Shari voice remains consistent
- authenticated preview passes the failure corpus
