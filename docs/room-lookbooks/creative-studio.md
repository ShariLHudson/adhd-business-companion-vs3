# Creative Studio — Room Look Book

**Place ID:** `creative-studio`  
**Status:** Design authority — Phase 1 (no UI implementation)  
**Registry:** [`lib/companionObjects/companionObjectRegistry.ts`](../../lib/companionObjects/companionObjectRegistry.ts)

---

## Purpose

The Creative Studio is where **making is allowed to be messy** — ideas visible, perfectionism forgiven, output invited but not policed.

It exists for **Create**, email drafting, and creative spark play — content and artifacts born in warmth, not a sterile content factory.

**Emotional need served:** "My ideas deserve room to breathe before they are judged."

**ADHD challenge reduced:** perfectionism paralysis, blank-page dread, shame about half-finished work, tool-hopping instead of shipping.

Sun-drenched room or enclosed porch — pegboard, jars, long worktable, golden hour ownership. Iowa sky on three sides when architecture allows.

---

## Emotional Promise

**When someone leaves this room…** they feel **more capable of making** — with something started or continued, not a lecture about consistency.

---

## Signature Feature

### The active project table

One hero only. A **long worktable with an active project in progress** — sketchbook open to a half-finished page, pencil dust, eraser crumbs, paint brushes in a jar, fabric scrap pinned, honest chaos that says *someone was already making here*.

Not a blank template gallery. Not a pristine Figma frame. The table says: *you can continue, not begin from zero*.

Golden hour light turns the table amber — permission in photons.

## Motion

Subtle only:

- Curtain lift in breeze
- Butterfly outside window (summer — rare, not circus)
- Lamp chain sway barely
- Shadow of oak branches across table
- Steam from nearby coffee cup (optional)

No confetti on save. No "great job!" particle burst.

---

## Companion Presence

**Present — Partner level.**

Shari works **alongside** making — co-writer, co-thinker, gentle challenger of perfectionism. She does not take the keyboard. She does not ship without consent.

Warmth + craft respect. She asks what good enough means here.

**Why not Host:** Creation is partnership, not welcome ceremony.

**Why not Absent:** Blank-table absence increases blank-page dread.

---

## Kinsey

**Occasionally present — not on the table.**

Kinsey may sleep on the studio rug away from pigments. She is **not** the hero.

- **Greeting:** Sniff, tail, exit — no jumping on project table
- **Not in this room:** During recording or hyperfocus sprint — Kinsey elsewhere

Never use Kinsey to interrupt flow for charm.

## UI Language

| Element | Rule |
|---------|------|
| **Cards** | Draft cards like paper on table — artifact type labeled in plain language |
| **Buttons** | "Continue draft," "Save," "Not ready" — warm solid |
| **Spacing** | Table-scale; one artifact focal; catalog picker grid only when explicitly choosing type |
| **Overlay** | ~25–35% hero (active table) then working layer dominant |
| **Typography** | Maker legibility; preview text book-like |
| **Glass** | Cream workspace — not glass morphism on golden hour photo |
| **Interaction** | Permission before workspace open; draft persists visibly |

Create catalog uses object placeholders from registry — never emoji tiles.

---

## Never In This Room

- Blank white SaaS template prison
- "You haven't published in X days" guilt
- Content calendar grid as default view
- Stock photo hero banners
- Neon growth-hacker CTAs
- Auto-posting without explicit consent
- Six-tool feature mall on the same surface
- Clip-art icons
- Kinsey or confetti stealing focus from the active project

---

## ADHD Rules

| Challenge | How the room helps |
|-----------|-------------------|
| **Executive function** | Active project visible — continue beats start |
| **Working memory** | Draft on table — externalized WIP |
| **Overwhelm** | One artifact type chosen; catalog collapsed until needed |
| **Decision fatigue** | Shari helps define "good enough to ship" |
| **Attention** | Golden hour + single table focal |
| **Emotional regulation** | Mess forgiven; perfectionism named gently |
