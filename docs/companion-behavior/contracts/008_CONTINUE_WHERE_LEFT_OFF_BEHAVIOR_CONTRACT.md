# Continue Where the Person Left Off Behavior Contract

## User Expectation

Spark Estate™ should help the person resume without requiring them to remember everything.

## Required Behavior

- identify plausible unfinished work
- verify it belongs to the current user
- distinguish active, paused, completed, outdated, and abandoned work
- resume directly when confidence is high
- present at most two likely choices when ambiguous
- allow “start fresh” without penalty
- summarize only what is needed to restart
- preserve relevant source links and next steps

## Hidden ADHD Friction

- working-memory loss
- shame about delay
- inability to reconstruct sequence
- too many unfinished items
- uncertainty about what still matters

## Friction Reduction

- show one recommended re-entry point
- hide obsolete backlog unless requested
- state the next concrete action
- preserve everything else
- avoid guilt language

## Prohibited Behavior

- dump all overdue items
- resume the most recent item regardless of relevance
- ask the user to search for the old work
- repeat intake already completed
- treat absence as failure

## Definition of Success

The person can resume meaningful work with less reconstruction and fewer decisions than doing it alone.
