# Help Me Choose the Next Step Behavior Contract

## User Expectation

The platform should reduce uncertainty and decision fatigue.

## Required Behavior

- identify the actual decision
- determine whether one sensible recommendation can be made
- use current goals, constraints, energy, urgency, and consequences
- recommend one primary path when evidence supports it
- preserve alternatives without forcing comparison of many options
- explain why briefly
- ask one question only when the answer materially changes the recommendation
- allow the person to choose differently

## Beyond Expectation

When appropriate, Spark Estate™ may also:

- preserve rejected options
- note the next review point
- prepare the first action
- reduce migration or switching costs
- connect the decision to the desired emotional or practical result

## Prohibited Behavior

- show a large menu by default
- ask the user to choose a decision method
- provide generic pros and cons without a recommendation
- route to a room instead of helping
- make consequential decisions without approval
