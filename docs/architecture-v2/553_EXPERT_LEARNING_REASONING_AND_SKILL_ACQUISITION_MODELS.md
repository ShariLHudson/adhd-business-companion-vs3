# 553 — Expert Learning Reasoning & Skill Acquisition Models

## Core sequence
1. define the real-world outcome
2. identify current ability
3. identify prerequisites and misconceptions
4. select the smallest useful capability
5. explain with an appropriate representation
6. model the skill
7. guide practice
8. reduce support gradually
9. test independent performance
10. reinforce over time
11. transfer into a new context

## Learning levels
- exposure
- recognition
- understanding
- recall
- guided performance
- independent performance
- flexible application
- teaching or coaching another person

Recognition is not mastery.

## Skill types
- declarative
- procedural
- conditional
- perceptual
- strategic
- interpersonal
- creative
- embodied
- metacognitive

## Error interpretation
An error may reflect missing knowledge, unclear instruction, working-memory overload, a weak prerequisite, insufficient practice, poor transfer, anxiety, fatigue, or inappropriate difficulty.

Do not interpret one failed method as inability.

## Confidence calibration
Help the user distinguish:
- I have seen this.
- I understand this.
- I can do it with help.
- I can do it independently.
- I can use it in a different situation.
