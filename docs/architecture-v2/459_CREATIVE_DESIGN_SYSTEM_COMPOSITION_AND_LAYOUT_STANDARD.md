# 459 — Creative Design System, Composition & Layout Standard

## Composition principles

- one dominant focal point
- clear reading path
- purposeful alignment
- controlled repetition
- sufficient breathing room
- intentional grouping
- meaningful contrast
- balanced visual weight

## Typography principles

- establish role-based hierarchy
- limit unnecessary styles
- preserve readable line length
- use adequate spacing
- avoid tiny decorative copy
- avoid all-caps paragraphs
- keep type editable when practical
- test at final display size

## Color principles

- apply the canonical palette
- use color to support hierarchy
- preserve adequate contrast
- do not rely on color alone
- test across screen and print
- avoid unapproved palette expansion

## Imagery principles

- support the message
- preserve realistic representation
- avoid misleading context
- maintain consistent treatment
- protect anatomy and identity
- avoid clichés when specificity is possible

## Reusable family requirements

Define:

- margins
- grid
- safe zones
- type roles
- spacing scale
- image behavior
- logo placement
- footer behavior
- call-to-action placement
- channel variants

## Responsive principle

Adaptation is not resizing alone.
Recompose for the destination format.
