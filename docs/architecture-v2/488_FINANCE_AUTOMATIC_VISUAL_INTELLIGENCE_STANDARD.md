# 488 — Finance Automatic Visual Intelligence Standard

## Purpose

Automatically create the most useful financial visual for the current decision.

## Supported visuals

- cash-flow timeline
- runway view
- revenue-cost-profit bridge
- break-even chart
- pricing comparison
- scenario range
- financial-health scorecard
- expense composition
- margin waterfall
- investment tradeoff map
- assumption sensitivity map
- decision threshold view

## Default display

Show only:

- current financial question
- one key number
- primary risk
- relevant range
- next review point

## Visual truth rules

- label estimates
- show uncertainty
- avoid misleading axes
- do not hide negative values
- distinguish cash from profit
- distinguish actual from forecast
- include source period

## Ownership

The shared visual engine owns rendering and interaction.

Finance Intelligence owns the financial meaning, calculation requirements, and decision relevance.
