# 456 — Creative Studio Naming, Ownership & Non-Duplication Contract

## Critical naming distinction

### Creative Studio

The Chamber member responsible for visual-design reasoning and creative production.

### Create Studio

The shared purpose-first creation destination where users create many kinds of outputs.

### Creative Studio Estate Place

The immersive visual location associated with creative work.

These may connect, but they are not interchangeable.

## Creative Studio owns

- visual communication
- hierarchy
- layout
- composition
- typography application
- color application
- imagery
- graphics
- presentation design
- branded asset production
- design adaptation
- visual accessibility
- creative-production specifications
- critique and preflight
- consistency across an asset family

## Universal Create owns

- shared creation runtime
- discovery
- session state
- saving
- resuming
- versioning framework
- output identity
- export orchestration
- Create-to-Project handoff

## Content & Communication owns

- wording
- narrative
- message structure
- teaching logic
- copy
- editorial quality

## Brand authority owns

- canonical identity
- approved palette
- approved typography
- logo rules
- brand-system truth

Creative Studio applies the system but does not silently redefine it.

## Marketing owns

- campaign objective
- audience strategy
- positioning
- channel purpose
- acquisition strategy

## Data & Analytics owns

- measurement validity
- performance analysis
- testing methodology
- reporting

## Technology owns

- rendering infrastructure
- export implementation
- media pipelines
- storage
- performance
- integrations

## No duplication rule

Creative Studio must never become:

- a second Create runtime
- a duplicate Content generator
- a second Brand authority
- a campaign strategist
- a file manager
- a generic image-prompt box
