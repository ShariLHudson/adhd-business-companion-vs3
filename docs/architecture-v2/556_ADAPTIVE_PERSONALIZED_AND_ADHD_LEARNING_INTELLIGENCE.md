# 556 — Adaptive, Personalized & ADHD Learning Intelligence

## Adaptation dimensions
Prior knowledge, working memory, processing speed, energy, attention, confidence, interest, sensory preference, urgency, available time, emotional state, modality, repetition need, and ambiguity tolerance.

## ADHD-aware defaults
- show the purpose first
- create a visible finish line
- reduce simultaneous instructions
- offer one next action
- use relevant examples
- allow verbal and movement-based processing
- preserve unfinished work
- support re-entry
- make progress visible
- use brief review before new material
- separate memory difficulty from intelligence

## Adaptive states
**Low energy:** reduce scope, shorten practice, provide examples.  
**Interested but scattered:** capture side questions and protect the target.  
**Frustrated:** identify the breakdown and restore support.  
**Overconfident:** test application without prompts.  
**Anxious:** reduce evaluative language and begin with a supported win.

## No fixed learning-type rule
Do not label users as permanently visual, auditory, kinesthetic, slow, or incapable.

Use observed patterns:
> “You seem to retain this better when we pair a short explanation with a concrete example and then let you try it.”
