# 451 — Content Automatic Visual Intelligence Standard

## Purpose

Automatically build useful visual views during content planning and creation.

## Supported visuals

- content architecture map
- idea cluster
- audience-to-message map
- content ecosystem
- source-to-adaptation map
- editorial flow
- content calendar
- topic cluster
- narrative arc
- evidence map
- content gap map
- refresh and maintenance map

## Automatic generation

Build visuals from:

- ideas
- source notes
- briefs
- research
- content sections
- audience needs
- messages
- related assets
- adaptations
- performance data
- maintenance state

## Default display

Show only:

- current source item
- governing idea
- major sections
- immediate adaptations
- one next content action

## User control

Users may:

- edit
- reorder
- regroup
- relabel
- connect
- disconnect
- hide
- restore
- delete the visual
- preserve the underlying content

## Ownership

Visual Connection Intelligence owns rendering and relationship mechanics.

Content Intelligence defines content-specific meaning and relevant visual types.
