# 571 — Marketing Automatic Visual Intelligence Standard

## Supported visuals

- market landscape
- audience segmentation map
- positioning map
- category map
- message hierarchy
- customer journey
- campaign architecture
- funnel flow
- lifecycle map
- channel portfolio
- content ecosystem
- experiment map
- KPI tree
- attribution map
- campaign timeline
- growth loop
- offer ladder

## Automatic visual triggers

Consider a visual when:

- multiple audiences differ
- positioning tradeoffs matter
- journey stages are unclear
- channel roles overlap
- campaign dependencies exist
- conversion drop-off matters
- messages form a hierarchy
- experiments need comparison
- growth loops interact
- the user needs to see the whole ecosystem

## Truth rules

Visuals must:

- distinguish evidence from hypothesis
- label estimated attribution
- show missing data
- avoid implying causation without support
- show current versus proposed states
- preserve audience and channel context

## Visual restraint

Prefer one decision-useful visual over decorative dashboards.

Marketing owns meaning, segmentation, position, journey, and measurement interpretation.

The shared visual engine owns rendering, interaction, and export.
