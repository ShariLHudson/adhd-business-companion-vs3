# 415 — AI / Intelligence Create Capability Catalog

## Purpose

Define the artifacts AI / Intelligence can help create without duplicating Universal Create.

## Capability categories

### AI use-case design

- AI opportunity brief
- use-case scorecard
- readiness assessment
- build/buy/integrate analysis
- pilot plan
- adoption roadmap

### Prompt and workflow assets

- prompt specification
- reusable prompt template
- AI workflow design
- agent workflow map
- human-review protocol
- escalation procedure

### Evaluation

- test plan
- evaluation rubric
- benchmark scenario set
- red-team checklist
- quality scorecard
- model comparison brief

### Governance

- AI acceptable-use guide
- data handling guide
- risk register
- oversight plan
- approval policy
- vendor review checklist

### Education

- AI literacy guide
- team training outline
- role-specific AI playbook
- glossary
- adoption FAQ

### Research

- AI trend brief
- product comparison
- model capability brief
- emerging-technology impact report

## Promise levels

Each artifact must be marked:

- fully creatable
- creatable with user information
- preparatory
- external completion required

## Automatic visual types

- workflow map
- decision map
- risk-control map
- evaluation matrix
- adoption roadmap
- agent flow
- capability landscape

## Ownership

Universal Create owns generation, save, edit, resume, and adaptation.

AI / Intelligence owns:

- domain logic
- required fields
- reasoning criteria
- risk constraints
- quality rules
