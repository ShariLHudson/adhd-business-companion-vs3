# 498 — Horizon Assumption, Memory, Indicator & Lineage Registry

## Purpose

Preserve future reasoning so scenarios can be reviewed and improved over time.

## Remember

- time horizon
- assumptions
- drivers
- weak signals
- counter-signals
- scenario versions
- early indicators
- decision implications
- chosen no-regret moves
- review dates
- actual developments
- lessons

## Do not store as permanent truth

- “This trend will definitely happen.”
- “This market is dead.”
- “AI will replace everyone.”
- “Customers will always want this.”
- “The future is obvious.”

Store evidence, date, uncertainty, and conditions.

## Indicator registry

Each indicator should include:

- signal
- source
- expected direction
- threshold
- time window
- scenario relationship
- review cadence
- owner
- current status

## Lineage

Every foresight artifact should reference:

- source evidence
- assumptions
- scenarios
- decisions influenced
- later updates
- superseded versions
