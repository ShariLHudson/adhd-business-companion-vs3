# 523 — Innovation Memory, Assumption, Lineage & Learning Registry

## Purpose

Preserve what was learned so failed experiments and retired concepts remain valuable.

## Remember

- opportunity
- problem evidence
- concept versions
- assumptions
- experiments
- success criteria
- results
- contradictory evidence
- decisions
- reasons
- costs
- participant context
- next review
- transfer destination

## Assumption classes

- problem
- audience
- value
- behavior
- feasibility
- pricing
- delivery
- adoption
- economics
- scale
- compliance

## Do not store as permanent truth

- “Customers do not want this.”
- “The idea failed.”
- “This market is impossible.”
- “The team cannot innovate.”

Store:

- what was tested
- with whom
- when
- under what conditions
- what happened
- confidence
- limitation

## Lineage

Every innovation artifact should connect:

- originating signal
- opportunity
- concept
- assumptions
- experiments
- evidence
- decision
- later implementation
- outcomes
