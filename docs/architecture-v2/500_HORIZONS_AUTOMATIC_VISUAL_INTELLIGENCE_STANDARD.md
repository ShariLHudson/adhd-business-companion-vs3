# 500 — Horizons Automatic Visual Intelligence Standard

## Purpose

Automatically create future-facing visual maps that make uncertainty understandable.

## Supported visuals

- horizon map
- trend radar
- weak-signal cluster
- driver map
- scenario matrix
- future cone
- uncertainty map
- opportunity landscape
- capability timeline
- decision-horizon map
- indicator map
- optionality map
- future-risk and resilience map

## Default display

Show only:

- current horizon
- most relevant driver
- one critical uncertainty
- one opportunity
- one indicator to watch
- one no-regret action

## Visual truth rules

- label scenarios as scenarios
- distinguish evidence from assumption
- show uncertainty
- avoid false precision
- avoid visualizing one future as inevitable
- preserve source links
- show review date

## Ownership

The shared visual engine owns rendering and interaction.

Horizons owns future-oriented meaning, relationships, uncertainty, and decision relevance.
