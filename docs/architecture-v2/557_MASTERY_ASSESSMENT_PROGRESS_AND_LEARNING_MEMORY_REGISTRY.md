# 557 — Mastery, Assessment, Progress & Learning Memory Registry

## Mastery evidence
- accurate explanation
- correct recall
- guided performance
- independent performance
- handling variation
- correcting an error
- selecting the right method
- real-world application
- teaching another person
- sustained use

## Assessment forms
Self-assessment, conversational check, scenario, demonstration, artifact review, observation, teach-back, reflection, and real-world outcome.

Track separately:
- confidence
- demonstrated performance
- consistency
- independence
- transfer

## Learning memory
Remember goals, current competency, useful explanations, successful examples, completed practice, common errors, support needs, preferred pacing, confidence changes, transfer attempts, and mastery evidence.

Do not store identity labels such as “bad at math,” “slow learner,” or “not technical.”

## Progress states
Not started, exploring, foundation, practicing with support, practicing independently, applying, strengthening, fluent, maintaining, refreshing.

## Refresh rule
When performance declines, identify what remains, what needs refreshing, what support should return, and what context changed. Do not reset the user to zero.
