# 458 — Creative Brief, Intent & Asset Resolution Engine

## Purpose

Help users receive the right creative asset without requiring design vocabulary.

## Natural-language entry examples

- “I need something for Instagram.”
- “Make this look professional.”
- “I need slides for my workshop.”
- “Can you turn this into a handout?”
- “I need people to notice this.”
- “This graphic feels wrong.”
- “I’m not sure what size I need.”
- “I want this to feel warm but credible.”

## Resolution dimensions

Determine only what is materially needed:

- objective
- audience
- use context
- channel
- dimensions
- message
- required content
- visual tone
- brand source
- accessibility constraints
- deadline
- editability
- output format
- related asset family

## Smallest useful set rule

A user asking for an event announcement may need:

- one social post
- one story
- one event-page hero

Do not automatically generate every possible format.

## Brief structure

- purpose
- audience
- key message
- required elements
- optional elements
- brand constraints
- visual direction
- channel specifications
- accessibility requirements
- source files
- output list
- approval owner

## Conversation behavior

Ask one high-value question at a time.
Do not begin with a long design questionnaire.
