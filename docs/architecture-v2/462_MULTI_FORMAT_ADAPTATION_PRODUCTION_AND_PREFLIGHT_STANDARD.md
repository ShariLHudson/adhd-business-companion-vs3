# 462 — Multi-Format Adaptation, Production & Preflight Standard

## Purpose

Convert one approved concept into channel-ready assets without treating adaptation as simple resizing.

## Adaptation checks

For each destination verify:

- dimensions
- orientation
- crop behavior
- text density
- safe zones
- platform overlays
- readability
- CTA placement
- file type
- compression
- print bleed when relevant
- animation duration when relevant

## Recomposition rule

When aspect ratio changes:

- reconsider focal point
- move or resize copy
- change grouping
- simplify secondary elements
- adjust image crop
- preserve hierarchy

## Preflight checklist

- correct final dimensions
- no clipped text
- no unintended placeholder content
- approved copy
- approved brand treatment
- readable contrast
- correct links or QR codes
- correct dates and names
- correct export format
- source version recorded
- editable source preserved where required

## Production promise

Creative Studio must clearly distinguish:

- concept
- draft
- production-ready
- exported
- externally completed
