# 318 — Cursor Completion Report Template

## 1. Executive summary

- work completed
- current architecture state
- major consolidations
- remaining risks
- production readiness

## 2. Authoritative owners confirmed

For each concept list:

- owner
- extensions
- projections
- adapters
- deprecated alternatives

## 3. Duplicate systems found

For each duplicate:

- path
- type
- impact
- selected disposition
- migration status

## 4. Repository changes

List:

- files added
- files changed
- files deprecated
- schemas changed
- tables changed
- APIs changed
- routes changed
- registries extended

## 5. Data migration

Include:

- records migrated
- records skipped
- orphaned records
- failed migrations
- rollback status
- validation counts

## 6. Blueprint retrofit status

For each Blueprint report:

- Work Type identity
- context mapping
- output mapping
- capability mapping
- shared-object mapping
- routing
- exact resume
- certification
- known limits

## 7. Tests

Include:

- tests added
- tests passed
- tests failed
- tests blocked
- evidence location

## 8. Certification

Report:

- Universal Work Type Certification
- Blueprint Createability
- Blueprint Context Connection
- Shared Object compliance
- No-Duplication Certification
- accessibility
- privacy
- domain certification

## 9. Known limitations

State clearly:

- unavailable features
- temporary adapters
- incomplete migrations
- future work
- risks

## 10. Unresolved decisions

For each:

- issue
- competing options
- recommendation
- impact
- decision needed

## 11. Rollback

Include:

- rollback steps
- migrations to reverse
- backups required
- validation after rollback

## 12. Final recommendation

Use one:

- ready for production
- ready for limited release
- ready with declared limits
- not ready
- blocked
