# 423 — Client Relationship Memory & Meaning Standard

## Purpose

Preserve relationship intelligence without turning people into static profiles or hidden scores.

## May remember

- stated communication preferences
- preferred level of detail
- decision style
- important goals
- concerns
- commitments
- approved personal context
- recurring frustrations
- appreciation preferences
- important dates
- previously successful approaches

## Must not infer permanently

- personality diagnosis
- emotional condition
- trustworthiness
- loyalty
- intent
- protected characteristics
- fixed relationship score

## Meaning memory rule

Store the observation and source, not an absolute label.

Good:

> “In the June planning call, Jordan asked for written options before making decisions.”

Avoid:

> “Jordan is indecisive.”

## User control

Users must be able to:

- view
- correct
- delete
- limit
- approve
- decline

relationship memory.

## Privacy

Relationship information must not be reused outside its appropriate context without permission.
