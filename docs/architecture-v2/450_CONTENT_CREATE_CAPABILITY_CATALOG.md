## Promise levels

Each capability must be classified as:

- fully creatable
- creatable with user information
- preparatory
- external completion required

- purpose
- audience
- format
- channel
- source material
- voice source
- evidence requirements
- primary owner
- related campaign or Project
- source artifact
- adaptation status
