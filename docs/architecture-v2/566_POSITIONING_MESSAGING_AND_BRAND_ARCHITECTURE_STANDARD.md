# 566 — Positioning, Messaging & Brand Architecture Standard

## Positioning structure

Every positioning decision should define:

- target audience
- relevant market or category
- primary problem or desired progress
- current alternatives
- differentiated value
- evidence
- tradeoffs
- reasons to believe
- situations where the offer is not the best fit

## Messaging hierarchy

- core promise
- primary value
- supporting benefits
- proof
- differentiators
- objections
- calls to action
- audience adaptations
- channel adaptations

## Message-quality tests

A message should be:

- relevant
- specific
- believable
- differentiated
- understandable
- emotionally appropriate
- evidence-aware
- consistent with the actual experience

## Brand architecture

Define:

- parent brand
- product or service brands
- offers
- sub-offers
- programs
- audiences
- endorsed relationships
- naming logic
- message relationships
- migration rules

## Anti-generic rule

Avoid:

- “We help you succeed.”
- “Transform your business.”
- “Unlock your potential.”
- claims that could describe almost any competitor

## Truth standard

Do not use:

- unsupported superlatives
- fabricated scarcity
- false urgency
- invented testimonials
- misleading comparisons
- promises beyond the actual offer
