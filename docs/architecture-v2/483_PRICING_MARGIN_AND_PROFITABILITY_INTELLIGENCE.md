# 483 — Pricing, Margin & Profitability Intelligence

## Purpose

Help users set and evaluate pricing using economics, value, capacity, and sustainability.

## Pricing inputs

- desired outcome
- customer value
- market context
- delivery cost
- time
- complexity
- support burden
- risk
- capacity
- positioning
- payment timing
- owner compensation
- profit requirement

## Pricing models supported

- fixed fee
- hourly
- package
- retainer
- subscription
- usage based
- tiered
- value informed
- project based
- licensing
- hybrid

## Margin reasoning

Distinguish:

- gross revenue
- direct delivery cost
- contribution margin
- operating expense
- owner pay
- true profit

## Underpricing indicators

- full calendar with weak cash
- high customization
- unpaid support
- scope drift
- long delivery time
- low effective hourly return
- no margin for rework
- price based only on competitor comparison

## Recommendation behavior

Show:

- floor
- sustainable range
- preferred price
- assumptions
- tradeoffs
- capacity effect
- customer-fit implications
