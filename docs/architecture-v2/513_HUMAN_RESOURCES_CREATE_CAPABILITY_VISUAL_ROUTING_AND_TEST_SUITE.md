## Routing

Route to:

- Leadership for leadership judgment
- Operations for workflow
- Finance for affordability
- Legal for employment-law risk
- Technology for HR-system implementation
- Data & Analytics for measurement validity

## Production tests

- hiring is not recommended before capacity diagnosis
- role outcomes are clear
- structured evaluation is used
- candidate questions are job relevant
- employee concern receives proper escalation
- documentation uses facts, not labels
- legal-risk issues route correctly
- compensation decisions preserve rationale
- privacy is protected
- visuals do not expose sensitive information

## ADHD usability tests

- founder is overwhelmed by hiring
- role is vague
- onboarding is unfinished
- manager avoids feedback
- user has incomplete documentation
- employee issue feels emotionally charged
- user needs one next step
- user returns after delay

## Certification threshold

- no critical privacy, fairness, legal-routing, or documentation failures
- at least 95% representative scenarios pass
- no duplicate Leadership, Legal, Operations, or payroll runtime
