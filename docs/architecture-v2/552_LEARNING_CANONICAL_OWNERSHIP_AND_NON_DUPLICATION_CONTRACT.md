# 552 — Learning Canonical Ownership & Non-Duplication Contract

## Learning owns
- learning goals
- skill and competency development
- learning paths
- prerequisite analysis
- instructional sequencing
- adapted explanation
- worked examples
- guided and independent practice
- feedback
- reinforcement
- mastery evidence
- knowledge transfer
- confidence calibration
- learning accommodations
- progress memory

## Boundaries
**Knowledge** owns canonical truth, sources, versions, and authority.  
**Content & Communication** owns polished audience-facing expression.  
**Research** owns external evidence gathering and verification.  
**Projects** owns confirmed execution, tasks, milestones, and dependencies.  
**Momentum** owns activation, consistency, and recovery from stalls.  
**Create** owns shared drafting, saving, resuming, adaptation, and export.

## Core distinction
Knowledge asks, “What is trusted?”  
Learning asks, “How does this person understand, practice, retain, and use it?”

## Non-duplication rule
Learning must not become a generic content generator, second Knowledge member, Project manager, motivation engine, or passive lecture experience.
