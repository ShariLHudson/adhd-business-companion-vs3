# 567 — Customer Journey, Campaign, Funnel & Lifecycle Intelligence

## Journey stages

- unaware
- problem aware
- solution exploring
- offer evaluating
- deciding
- purchasing
- onboarding
- activating
- using
- renewing
- expanding
- advocating
- returning

Not every business requires every stage or a linear journey.

## Journey record

Each stage may include:

- customer question
- emotional state
- barrier
- trust need
- desired evidence
- message
- channel
- experience
- action
- measurement
- owner

## Campaign architecture

Every campaign should define:

- objective
- audience
- insight
- offer
- message
- channel mix
- journey stage
- creative needs
- timeline
- budget
- conversion event
- learning questions
- success criteria
- stop criteria
- handoffs

## Funnel intelligence

A funnel is a measurement model, not a complete customer experience.

Track:

- entry source
- stage progression
- drop-off
- time between stages
- quality
- intent
- conversion
- retention
- downstream value

## Lifecycle marketing

Lifecycle activity may support:

- activation
- education
- engagement
- retention
- renewal
- expansion
- advocacy
- reactivation

## ADHD-aware campaign design

For founders and users with ADHD:

- reduce simultaneous campaigns
- make the current campaign visible
- define one primary conversion
- preserve ideas in a backlog
- separate setup from ongoing maintenance
- create re-entry summaries
- show the next smallest marketing action
