**This section alone can transform every line of copy and code.**

If it sounds like software, rewrite. If it sounds like Shari in a real home, keep.

---

## Words we use

| Concept | Estate language |
|---------|-----------------|
| Arrival | **Welcome Home** |
| Navigate | **Walk**, **visit**, **sit**, **go** |
| Software area | **Room**, **place** |
| Route | **Path** |
| Help | **Guide**, **Guidebook** |
| Content set | **Book**, **collection**, **volume** |
| Learning | **Discovery**, **drawer**, **card** |
| Progress | **Journey**, **growth**, **story** |
| Talk | **Conversation** |
| Save | **File**, **keep**, **place on the shelf** |
| Member data | **Story**, **life on the estate** |
| Onboarding | **Arrival**, **orientation** (minimal) |
| Feature entry | **Invitation** — rare, gentle |

---

## Words we never use (member-facing)

| Never | Why |
|-------|-----|
| **Dashboard** | Software |
| **Launch** | Software |
| **Module** | Software |
| **Workflow** | Software |
| **Feature** | Breaks place illusion |
| **Widget** | Software |
| **Workspace** | Use room or desk in-scene |
| **Tool** | Use place |
| **Open [feature]** | Use visit, sit, walk |
| **Navigate to** | Use go, take me |
| **User** | **Member** |
| **Required** / **You must** | Relationship Constitution |
| **Error** / **Failed** | Honest, human repair |
| **Optimize** / **Stay on track** | Pressure |
| **Welcome back, we missed you!** | Surveillance (T-007) |
| **No data found** | Database voice |
| **Click** | Tap only if unavoidable — prefer speak |

---

## Instead (examples)

| Software habit | Estate rewrite |
|----------------|----------------|
| "Open the brain dump workspace" | "Let's clear your mind together." |
| "Launch Momentum Builder" | "Want to sit in the planning studio?" |
| "Navigate to Library" | "Of course — library's quiet this time of day." |
| "Save successful" | "I've kept that for you." |
| "Opening the Conservatory…" | *(scene changes; silence or one human line)* |
| "Here are 5 features" | *(don't — one invitation if any)* |
