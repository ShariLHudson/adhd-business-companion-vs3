## Object principles

1. **Owned, not programmed** — member feels possession  
2. **Story first** — every object has history or purpose  
3. **Open and close** — return to scene; never app takeover  
4. **Permission** — save, file, publish in room language  
5. **Nothing decorative-only** — if it appears, it matters (Article XIII)
