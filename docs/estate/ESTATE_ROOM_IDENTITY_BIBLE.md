## How to use this bible

Before any design decision, ask:

1. Does this honor the **core emotional feeling**?
2. Does it serve the **primary purpose** through the **dominant verb**?
3. Does it feel like the **metaphor** and **material**?
4. Do **light**, **color**, **sound**, and **pace** match?
5. Is this how **Spark** would speak in this room?
6. Would the **signature sentence** still be true?

If no → redesign the choice, not the room’s soul.

# Journal Gazebo

| Attribute | Identity |
|-----------|----------|
| **Core emotional feeling** | Peaceful permission — you may think out loud without an audience |
| **Primary purpose** | Capture thoughts, reflections, emotions, ideas, and life as it happens |
| **Dominant verb** | **Rest** *(thoughts rest on the page)* |
| **Primary metaphor** | A **white gazebo** with an open notebook on the railing |
| **Primary material** | Handmade paper, linen, weathered wood, ink |
| **Lighting** | Soft daylight — dappled, diffuse, never harsh; late-morning stillness |
| **Color mood** | Warm ivory, parchment, soft garden green, whisper of teal; no cold grey |
| **Soundscape** | Distant birds, a breeze through leaves, near-silence; no music unless member brings it |
| **Pace** | Unhurried — the visitor sets the tempo; long pauses are welcome |
| **Spark's personality** | Gentle, patient, stepping back; a companion who opens the door and lets you write |
| **Signature keepsake** | A **pressed page** — photo or image tucked into the margin of a memory |
| **Signature sentence** | *"A place where scattered thoughts become clearer."* |
| **Signature transition — entering** | *Crossing into quiet light — the noise of the Estate softens; the page is already waiting.* |
| **Signature transition — leaving** | *The page is closed; the words remain. You step back into the Estate lighter than you arrived.* |

**Identity guardrails:** Never performative. Never analytics. Never “productivity journal.” Writing is the hero.

# Growth Greenhouse

| **Primary purpose** | Record what is still developing in life and work |
| **Dominant verb** | **Tend** *(place and nurture; never force)* |

| **Primary material** | Glass, clay pots, rich soil, condensation, handwritten plant markers |

| **Soundscape** | Muffled rain on glass (optional, distant), humid hush, no ticking clocks |
| **Pace** | Seasonal — weeks between visits are normal; dormancy is not failure |
| **Spark's personality** | Patient gardener — curious about seasons, never asks “are you done yet?” |
| **Signature keepsake** | A **marker stake** — a note or image of what was planted that day |
| **Signature sentence** | *"A place where becoming is allowed to take its time."* |
| **Identity guardrails:** Never project management. Never progress bars. Never streaks. Organic only.

## Spark personality across rooms

Spark is always the **Estate Concierge** — but voice weight shifts with the room:

| Room | Spark is more… | Spark is less… |
|------|----------------|----------------|
| Journal | Present, then silent | Instructional, analytical |
| Greenhouse | Patient, seasonal | Urgent, metric-driven |
| Vault | Steady, grounding | Cheerful, casual |
| Library | Curator, admiring | Promotional, resume-coach |
| Garden | Warm, delighted | Childish, hyped |
| Hall | Reverent, sparse | Chatty, celebratory |

**Universal Spark rules in all rooms:** permission before action · no nagging · no surveillance language · Shari test on every line.

## Transition language — canonical patterns

**Entering** — atmosphere shift + threshold (never “Opening Journal…”).

**Leaving** — what remains + visitor carries something (relief, proof, pride, joy, honor).

| Room | Enter (short) | Leave (short) |
|------|---------------|---------------|
| Journal | Quiet light; page waiting | Words remain; step back lighter |
| Greenhouse | Warm glass; slow allowed | Tended thing stays in light |
| Vault | Still air; weight deserves keeping | Capability stays with you |
| Library | Paper and time; shelves hold stillness | Volume not lost; library keeps watch |
| Garden | Sweeter air; path opens | Planted bloom remains |
| Hall | Pace slows; not everyday | Chapter kept; you carry it |

## Identity conflicts — how to resolve

| Tension | Resolution |
|---------|------------|
| Feature vs atmosphere | Atmosphere wins |
| Information density vs calm | Calm wins |
| Multiple rooms at once | Identity of P1 room governs voice; second room waits |
| Empty vs abandoned | Empty = invitation per room metaphor |
| Browse vs create | Journal/Greenhouse: create identity; Library/Garden return: browse identity |
| Software copy vs room soul | Rewrite until Shari test passes |

- [ ] **Spark personality** matches room row in table above
