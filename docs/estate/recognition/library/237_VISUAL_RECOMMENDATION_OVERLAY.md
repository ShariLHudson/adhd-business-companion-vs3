## Purpose

Recommend visual thinking without interrupting conversation.

---

## Behavior

Spark offers recommendations only when they improve clarity.

**Example**

> I think a Timeline would make this easier to understand.

**Options**

- Create Timeline
- Show Other Maps
- Not Now

Recommendations should be subtle, dismissible, and never block progress.
