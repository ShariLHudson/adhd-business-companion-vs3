# Visual Discovery Interview™ Library

## Purpose

Every visual method has its own Discovery Interview™.

Spark should never use the same interview for every map.

The map determines the interview.

---

# Universal Rules

Before asking questions, Spark checks:

- Current conversation
- Clear My Mind™
- Journal
- Projects
- Uploaded files
- Previous maps

Ask only for missing information.

Stop asking questions as soon as Spark has enough information to build an excellent first draft.

---

## Mind Map

Discover:

- Main topic
- Major ideas
- Natural groups
- Relationships
- Desired outcome

Example questions:

- What are we mapping?
- What ideas immediately come to mind?
- Is there an end goal?

---

## Decision Map

Discover:

- Decision
- Options
- Constraints
- Risks
- Success criteria

---

## Strategy Map

## Relationship Map

The Discovery Interview changes.
