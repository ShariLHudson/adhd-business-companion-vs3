## Purpose

Ensure visual work is never lost.

## Save Behavior

Automatically preserve:

- Nodes
- Connections
- Notes
- Selected view
- Source context
- Last edit time

## History

Members should be able to:

- Resume a previous map
- Duplicate a map
- Rename a map
- Archive a map
- Delete a map
- Restore a recently deleted map

## Integration

Completed maps may be sent directly to:

- Projects™
- Destination Gallery™
- Google Docs
- Google Drive
- Journal
- Evidence Vault™

## Success

Members can leave a visual workspace and confidently return exactly where they left off.
