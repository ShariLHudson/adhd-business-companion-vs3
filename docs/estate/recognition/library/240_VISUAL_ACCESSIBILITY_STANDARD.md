## Principles

Every visual experience must be accessible.

---

## Requirements

- Voice-first support
- Full keyboard navigation
- Screen reader labels
- Large touch targets
- High contrast option
- Adjustable text size
- Reduced motion mode

No visual feature should require precise mouse control.
