## Purpose

Define every interactive object in the room.

---

## Interactive Objects

### Framed Maps

- Hover: brass light brightens
- Click: open associated Discovery Interview
- Right click / touch hold: learn about this map

### Cartographer's Atlas™

- Opens the Atlas of Visual Thinking™
- Browse by category
- Search
- Favorites
- Recently used
- Create from page

### Discovery Table

- Displays active map
- Supports editing
- Supports transformations
- Auto-saves continuously

### Telescope

- Zoom out to see the "big picture"
- View related projects, goals, and connected maps

### Globe

- Browse all map types by category

### Exit Door

- Return to Spark Estate™
- Resume later automatically

## Rule

The room itself is the navigation.

Avoid traditional software menus whenever possible.
