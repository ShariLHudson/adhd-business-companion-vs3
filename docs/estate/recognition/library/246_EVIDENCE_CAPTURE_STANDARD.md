# Evidence Capture Standard

Spark should recognize moments worth saving.

Examples:

- Thank-you messages
- Client testimonials
- Finished projects
- Personal breakthroughs
- Positive journal entries
- Kind words
- Before/after stories
- Milestones

When detected ask:

"Would you like to save this?"

Options:
- Evidence Vault™
- Hall of Accomplishments™
- Both
- Not now

Evidence should require no manual organization.
