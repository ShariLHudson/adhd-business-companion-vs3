# Spark First Draft Intelligence™

## Purpose

Spark builds the first draft.

Members refine it.

Members should rarely begin with a blank canvas.

---

## Spark Thinking Order

1. Purpose
2. Themes
3. Natural groups
4. Relationships
5. Priorities
6. Missing pieces
7. Risks
8. Opportunities
9. Suggested next steps

---

## First Draft Requirements

Every draft should:

- Be logically grouped
- Remove duplicates
- Highlight gaps
- Suggest missing branches
- Explain why information was grouped together

Spark should say:

"I think this is a strong starting point. We can refine it together."

---

## Success

The member immediately feels understood.

The map should save thinking time instead of creating more work.
