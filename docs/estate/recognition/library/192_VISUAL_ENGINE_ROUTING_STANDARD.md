## Purpose

Define how information flows into and out of the Spark Visual Engine™.

## Sources

- Current conversation
- Clear My Mind™
- Projects™
- Journal
- Decision Compass™
- Evidence Vault™
- Destination Gallery™
- Uploaded documents
- Previous visual maps

## Destinations

Visual maps may create or update:

- Projects
- Tasks
- Calendar
- Journal
- Evidence Vault
- Destination Gallery
- Google Docs
- Google Drive

## Routing Rule

The Visual Engine never owns data.

It visualizes existing information and returns updates to the originating experience.
