## Purpose

The Visual Engine must understand what the member is working on before creating a visual representation.

## Context Sources

The engine may visualize information from:

- Current conversation
- Clear My Mind™ session
- Projects™
- Journal
- Decision Compass™
- Destination Gallery™
- Evidence Vault™
- Uploaded notes or documents
- Existing visual maps

## Context Rules

- Reuse existing context whenever possible.
- Never require the member to re-enter information already available.
- Preserve links back to the original source.
- Allow multiple sources to be merged into one visual workspace.

## Context Banner

Always show the source of the current map, for example:

Current Context:
• Project: Workshop Launch
• Source: Clear My Mind Session
• Created: Today

Members may change or clear the context at any time.
