## Purpose

Define every recognition path that should activate the Spark Visual Engine™.

## Direct Recognition

Immediately launch the requested view when members say things like:

- Create a mind map
- Make a workflow
- Show me visually
- Map this out
- Create a process map
- Show relationships
- Build a timeline
- Create a decision tree
- Brainstorm visually

Do not ask unnecessary follow-up questions.

## Indirect Recognition

Spark may recommend visualization when it detects:

- Overwhelm
- Numerous disconnected ideas
- Multi-step projects
- Complex decisions
- Systems thinking
- Planning requests

The recommendation is optional.

## Recognition Priority

1. Explicit member request
2. Existing active context
3. Spark recommendation
4. Manual Visualize button
