
# 116 — Estate Navigation Constitution

Mission:
Users should think about their work, not where it lives.

Principles:
- Conversation first.
- Intent over menus.
- Rooms reduce cognitive load.
- Navigation is reversible.
- Return-to-work is always available.

Preferred routing:
User intent → Shari → Correct room → Same Work identity.

No feature should require users to memorize the Estate.
