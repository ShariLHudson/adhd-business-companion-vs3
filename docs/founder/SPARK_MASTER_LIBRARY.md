## The Shari Rule

Every entry answers:

- Where is it?
- Why do I use it?
- When should I use it?
- What is it best for?
- How does it help Visual Spark Studios?

---

## Maintenance rule

When a new permanent document is created:

1. Register in `lib/sparkMasterLibrary/sample/libraryData.ts`
2. Add to **Founder Knowledge Vault** if executive-facing
3. Add row to **MASTER_LIBRARY_INDEX.md**
4. Add to **MASTER_PROMPT_LIBRARY.md** if it is a prompt

See `.cursor/rules/master-library.mdc`.
