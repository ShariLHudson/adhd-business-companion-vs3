# Spark Estate™ Onboarding and First 7 Days Experience Specification - Phase 23

## Purpose

This document defines the first experience a new member has with Spark
Estate™.

The goal:

Help the member quickly understand the value of Spark and experience an
early success.

The first days should create:

-   confidence
-   connection
-   understanding
-   momentum

------------------------------------------------------------------------

# Core Principle

Do not teach the entire system first.

Help the member experience the system.

The member should think:

"This understands me."

Not:

"I need to learn how this works."

------------------------------------------------------------------------

# Stage 1: Welcome

## Goal

Create connection.

The first interaction should introduce:

-   who Spark is
-   how Spark helps
-   what the member can expect

Example:

"Welcome. I'm here to help you turn ideas into action, organize what
matters, and keep moving forward."

------------------------------------------------------------------------

# Stage 2: Learn About the Member

Gather naturally:

## Identity

-   name
-   preferred name
-   language

## Goals

-   what are you working toward?
-   what matters most right now?

## Challenges

-   what feels hardest?
-   where do you need support?

## Preferences

-   how do you like guidance?
-   examples or explanations?
-   simple steps or deeper detail?

Avoid:

-   long questionnaires
-   overwhelming setup screens

------------------------------------------------------------------------

# Stage 3: First Success Moment

The member should complete something quickly.

Examples:

-   capture an idea
-   organize thoughts
-   create a next step
-   save a goal
-   complete a small task

The first win creates trust.

------------------------------------------------------------------------

# Stage 4: Introduce the Estate

Do not show every room.

Introduce gradually.

Explain:

"You can visit different spaces depending on what you are working on."

------------------------------------------------------------------------

# Stage 5: First Spark Card™

Introduce the daily companion.

Explain:

"Spark Cards are little moments designed to encourage, inspire, or help
you notice something useful."

------------------------------------------------------------------------

# First Week Journey

## Day 1: Connection

Goal:

Understand the member.

Experience:

-   welcome
-   profile basics
-   first small win

------------------------------------------------------------------------

## Day 2: Clarity

Goal:

Help organize thoughts.

Experience:

-   idea capture
-   priorities
-   next step

------------------------------------------------------------------------

## Day 3: Creation

Goal:

Help create something.

Experience:

-   use universal creation journey

------------------------------------------------------------------------

## Day 4: Progress

Goal:

Show movement.

Experience:

-   Momentum Card
-   progress review

------------------------------------------------------------------------

## Day 5: Personalization

Goal:

Learn preferences.

Experience:

-   adapt guidance

------------------------------------------------------------------------

## Day 6: Discovery

Goal:

Show helpful rooms.

Experience:

-   explore relevant capabilities

------------------------------------------------------------------------

## Day 7: Reflection

Goal:

Strengthen relationship.

Ask:

"What has been most helpful so far?"

"What would make Spark better for you?"

------------------------------------------------------------------------

# Onboarding Rules

Spark should:

-   be welcoming
-   reduce overwhelm
-   celebrate progress
-   guide gently
-   adapt to the person

Avoid:

-   feature tours
-   complicated explanations
-   forcing setup completion

------------------------------------------------------------------------

# Success Measurements

A successful first week means:

The member:

✓ returns

✓ creates something

✓ experiences progress

✓ understands the value

✓ feels supported

------------------------------------------------------------------------

# Final Vision

The first seven days should transform:

"I downloaded an app."

into:

"I found a companion that helps me move forward."
