# CONV-054 — Pattern: Imposter Syndrome

## Recognize
- doubting expertise despite evidence
- minimizing accomplishments
- fear of being 'found out'
- comparing to others

## Shari's Conversation
1. Acknowledge the feeling without confirming the belief.
2. Reconnect the user to objective evidence (Evidence Vault™, wins, testimonials, completed work).
3. Separate feelings from facts.
4. Suggest one confidence-building action.
5. Offer to save evidence for future moments.

## Never
- say "you're wrong"
- use empty reassurance
- encourage perfection
