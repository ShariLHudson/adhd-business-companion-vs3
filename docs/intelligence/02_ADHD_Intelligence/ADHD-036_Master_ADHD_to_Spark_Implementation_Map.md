# ADHD-036 — Master ADHD-to-Spark Implementation Map

## Purpose

Connect each ADHD friction domain to the Spark systems that prevent, support, and recover.

## Core Mapping

- Working memory → Memory, Search, Decision History, Shari Connector
- Initiation → Welcome Home, Tiny Start, Templates, Concierge
- Transitions → Calendar Concierge, Closeout, Reset, Resume
- Time blindness → Buffers, Now/Next/Later, Reminders
- Overwhelm → Clear My Mind™, Open-Loop Radar, reduced choices
- Emotional strain → Shari, Soundscapes, Journal, response pause
- Hyperfocus → Landing Pad, transition prompts
- Routines → Rhythms, flexible windows, recovery
- Project overload → Priority Protection, Parking Lot
- Relationships → Briefings, promises, follow-up
- Administration → Administrative Radar, preparation packets
- Recovery → Return Plan™, Return Home
- Sensory load → Environment presets, Quiet Estate™
- Capacity → Adaptive Day Versions

## Build Rule

Do not create a new feature when an existing capability can be connected, surfaced, or made conversational.
