# ADHD-029 — Prevention Intelligence Framework

## Purpose

Define how Spark helps prevent predictable ADHD friction before it becomes urgent.

## Prevention Cycle

1. Notice an early signal.
2. Check whether the interpretation is correct.
3. Explain the possible risk.
4. Offer a preventive action.
5. Ask permission.
6. Perform or guide the action.
7. Review whether it helped.
8. Learn only with permission.

## Prevention Categories

- Time and deadline prevention
- Overload prevention
- Project-sprawl prevention
- Communication and follow-up prevention
- Emotional escalation prevention
- Transition failure prevention
- Administrative emergency prevention
- Recovery avoidance prevention

## Rule

Prevention should feel like support, not surveillance.
