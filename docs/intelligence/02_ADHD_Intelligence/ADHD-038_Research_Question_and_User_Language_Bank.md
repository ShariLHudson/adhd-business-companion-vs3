# ADHD-038 — Research Question and User Language Bank

## Purpose

Capture the natural language users use so Spark can recognize needs without requiring feature names.

## Question Families

### Starting
- “Where do I begin?”
- “Can you just start it?”
- “Why can’t I make myself do this?”

### Remembering
- “Don’t let me forget.”
- “Where did I put that?”
- “Why did we decide this?”

### Time
- “How did the day disappear?”
- “Can this fit before my meeting?”
- “How long will this really take?”

### Overwhelm
- “My brain is full.”
- “Everything feels urgent.”
- “Can you sort this out?”

### Recovery
- “I fell off again.”
- “Do I have to catch up?”
- “What still matters?”

### Relationships
- “Who did I promise to contact?”
- “Can you help me reply?”
- “Did I mess this up?”

### Business
- “Which project should I protect?”
- “Am I chasing another idea?”
- “What should I prepare before launch?”

## Research Use

Founder Intelligence should continually add exact user wording, group similar questions, and map each phrase to the relevant Spark capability and safe response.
