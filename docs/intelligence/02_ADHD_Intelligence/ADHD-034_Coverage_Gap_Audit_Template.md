## Rating Scale

- 0: Not addressed
- 1: Mentioned only
- 2: Partial feature
- 3: Useful but disconnected
- 4: Strong end-to-end support
- 5: Preventive, adaptive, connected, and recovery-aware

## 12/10 Standard

A challenge is not fully addressed until Spark can help prevent it, support it, and help the user recover from it.
