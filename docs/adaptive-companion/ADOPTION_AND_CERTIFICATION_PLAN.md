- [ ] Reads canonical preference source / resolver

- [ ] Reduced-choice mode has More Options when truncated  
- [ ] Summary-first can open full detail  
- [ ] One-step mode preserves progress  
- [ ] Pause / resume retain context  
- [ ] Direct user requests override stored presentation  
- [ ] No diagnosis or unapproved inference  
- [ ] Tests cover default and adapted states
